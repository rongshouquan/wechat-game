## 0. 高概念一页（High Concept）

- **一句话**：Q 萌轻松、纯自动战斗、基地养成、星舰收集与阵容搭配的微信小游戏。
- **玩家是谁**：刚接手小破星港的"星港小队长"，在又乱又热闹的星域里修星港、救居民、收星舰、得星核，把小基地养成热闹大星港。
- **核心循环**：上线看基地变化 + 领离线/打捞 → 战前搭配（驾驶员/星舰/插件/星核 + 站位）→ 自动战斗推关 → 拿货币/碎片/插件 + 首通解锁 → 养成扩建 → 突破 Boss、推 3天行动/7天扩张 → 开宝箱得星核 → 更能打、更热闹 → 循环。
- **爽点引擎**：卡关 → 渴望更强 → 拿到牛星核 → 兵种质变·画面更爽 → 碾压推几关 → 再卡。
- **北极星**：**体验平滑不憋屈**——卡关像"今天先到这"，不像"被堵死"。
- **四层定位（本作骨架）**：星舰=打什么/底子 · 驾驶员=怎么打/打谁 · 插件=补多少数值 · 星核=发生什么质变。
- **题材基调**：Q 萌、明亮、可爱，不做压抑战争感。敌人=星盗/失控无人舰/废铁机械/星能污染舰（后者做 Boss）。
- **商业化**：首发纯广告（不卡死、不补战力），预留内购接口。
- **平台约束**：微信小游戏 / Cocos / 竖屏 / 主包 ≤4MB / 纯本地存档（预留云）。

## S0 设计北极星
**体验平滑不憋屈。** 卡关/收手像"今天先到这"、不像"被堵死"。长时段深度靠后续内容模式（竞技场/副本），**不靠主线无限刷**。凡涉等待/卡点都要回答"如何平滑不憋屈"。

## S1 定位与核心体验
一句话：**Q萌轻松、纯自动战斗、基地养成、星舰收集与阵容搭配**的微信小游戏。玩家=接手小破星港的**星港指挥官**，修星港/救居民/收星舰/得星核/养成热闹大星港。核心体验：上线看基地变大；纯自动战斗不手操；失败回"看战斗+搭配+养成"不回操作时机；同资源不同培养路线；星核=第一稀缺品；忙几天回来不焦虑。题材 Q萌明亮。敌人 4 族群（星盗/失控无人舰/废铁机械/星能污染舰）。

## S2 目标玩家与节奏目标
四类玩家（轻/普/重/肝）；**普通=平衡基准**（~35分钟/天、目标 D21 毕业近 5 核）。看广告不计入基础速度。**重点先保前 3-7 天体验**；21/28 天靠节奏表 + 上线真实数据调。
- **肝档单独锁定（Ron 2026-07-02·推翻旧"肝帝D14毕业"结论）**：肝档（2-3h+/天）**≥30天毕业**、且每天在线时段主线内容不断档。**毕业=主线关卡全清**（不再套死"5核"，终局养成门槛自定）。主要靠**关卡数量/深度上调 + 已首通关卡撞墙期可重刷小奖**（S8）撑住，不靠侧系统硬撑；难度曲线参考《放置军团》做"章节/星域末尾卡关1-2天"。具体关卡量/曲线待毕业节奏建模反推。

## S3 主循环

## S4 战斗系统

## S5 战斗单位模型（v2.0 修订 · 覆盖 v1.0 §5）

> v2.0 对四层模型的核心修订，**覆盖 v1.0 §5.1 的"6 武器派系"**。完整 20 星舰 + 20 驾驶员见 `GDD-附录D-星舰真源.md` / `GDD-附录D-驾驶员真源.md`，弹种见 `GDD-附录D-通用攻击视觉库.md`。

**5.1 四层定位（核心不变）**：战斗单位 = 驾驶员 + 星舰 + 插件 + 星核。星舰=动作本体（打什么）· 驾驶员=打谁/怎么打 · 插件=补数值 · 星核=规则质变。

**5.2 星舰 5 定位型（替代 6 武器派系）**：星舰按**定位**分 5 型——
- **突击舰**（刺客）：低血/高攻速/高单体爆发、收割。
- **护卫舰**（坦克）：高血高防、站前排吃伤。
- **炮击舰**（输出）：正面强火力压制。
- **支援舰**（辅助）：治疗 / 护盾 / 给友增益。
- **工程舰**（功能）：召唤单位/无人机 + 给敌方负面。
> 定位靠"属性 + 火力形态"体现，**星舰技能不指定打谁**（打谁由驾驶员定）；首发各型 4 艘、共 20。

**5.3 层级分工铁律**：**"造成/施加"归星舰，"利用/放大"归驾驶员**。
- 星舰技能 = 具体动作（开火/范围/群体/召唤/护盾/治疗/增益/**施加状态**），不写"对某类敌人增伤"。
- 驾驶能力 = 打谁/护谁（含嘲讽、守护等行为）；驾驶天赋 = 利用局面的签名规则（增伤/击杀触发/叠加/自保/放大本舰辅助），**绝不施加状态**。
- 判据：能画成一次开火/放招/召唤的=星舰；改交战规则/利用局面的=驾驶员 → 二者天然连招。

**5.4 作用形态 + 状态/控制库**：
- 形态：单体 / 小范围(十字5格) / 一片(3×3) / 一列(5格) / 一直线 / 全体 / 友方同理。**残血=血量<30%**。召唤物=同格·不移动·可被打掉·召唤源死则消失。
- 状态库：硬控=短路/沉默；软控减益=减速/虚弱/破防/致盲/易伤/燃烧/削盾/驱散；增益=加攻/加攻速/护盾/回血/增伤/净化。**平衡=硬控递减 + Boss 控制抗性**。控制由星舰施加、驾驶天赋利用。

**5.5 成长模板（三条线都有奔头 · Ron 2026-06-25 重排）**：
- **起手 Lv0（只普攻）**，Lv1 解锁第一个技能/天赋（→ 新手引导"第一场只普攻"成立）。
- **升级**：星舰每级长属性、驾驶员每级 **+1% 驾驶加成**；并 **每 20 级（Lv20/40/60/80/100）一个"升级大节点"强化**（数值/小机制，非质变）。
- **升阶/升星 = 质变线**：星舰 **A阶=大质变 / SS阶=终极质变**（C(1槽)→B(2槽)→A(3槽)→S(开星核槽)→SS、开等级上限 20/40/60/80/100、每阶全属性跳）；驾驶员 **3★=大质变 / 5★=终极质变**（能力线 +10%/星）。→ 攒碎片升 A 阶 / 升 5 星就质变变强，给足抽卡乐趣。
- **战力**：插件 + 星核 + 驾驶加成% + 升星能力% + 可量化天赋，全部计入（呼应台账"战力计入星核与插件"）。

## S10 各玩法系统（迁自 v1.0 §10 · 含抽卡碎片化/商人/背包改动）

### S10.1 抽卡（星港补给站）· 三池 + **碎片化（本轮大改）**
- **三池**：招募补给（驾驶员）/ 专属补给（专属·居中）/ 舰船补给（星舰）。招募/舰船每日各开 2 类、每天轮换、3 天一轮；专属池常驻全部非专属 + 1 艘当期轮换专属。
- **每池产出（碎片化）**：**大概率出"本池类型的随机专属碎片（暂定 1~3 个）"、小概率出本体（C/B/A 阶级）**。→ 大幅放慢毕业、支撑"每天爽抽"。
- **未拥有单位的专属碎片**：**存背包，攒够可在背包合成 该单位的 C 级星舰 / 1★ 驾驶员**（碎片解锁新单位的路径）。
- **保底**：**20 抽内必出 A 级本体**，但做**真概率**（每抽都有机会、可能 10 抽就出、甚至 10 抽双/三蛋黄），非死等第 20 抽。
- **同单位低阶 → 抽到高阶**：后台自动把低阶那只**分解成该单位专属碎片**（留高的；不同单位不互相分解）。
- 补给券不过期（攒券针对性抽）；主来源 = 商人每日固定供应 + 主线/Boss 首通 + 3天/7天活动 + **赞助补给（看广告得券·每日上限）**。
- 数值（各池出率/碎片量/合成阈值/保底真概率）→ `《数值细表真源》`。

### S10.2 打捞与信标
- 获信标 → 选信标/打捞队/时长 → 等待 → 带回。信标档定深度、时间定数量与"稀有发现"掷骰。**打捞 = 通用碎片(舰/员)除活动外的稳定来源**（每档保底 1 种）；**也产星矿**、星贝、星核碎片/星空宝石（按档）。三档（普通/稀有/史诗）；时间档 2h/8h/24h；居民/工人/星辉货舱/各品质插件单次≤1。打捞队 1→3（打捞港升级），不会失败。
- **加速两种方式**：① 看广告（免费·每日上限）；② **「打捞加速券」**（星贝在商人小站买，分普通/稀有/史诗三档，**各在对应品质信标的打捞中减 2 小时、同一次打捞可多次使用**）。加速券**不设每日次数上限**，靠 **"券价偏高 + 星贝产出受控"自然节制**——前期信标少、不划算（买了反拖发育），定位为**中后期星贝富裕时"花星贝买时间"**的去处。加速=缩短剩余等待，到点照常结算原定收益。

### S10.3 商人小站（星贝）
- 定位：item sink + 中端定向购买 + 逛店惊喜。回收（信标/溢出星矿 → 星贝，随等级解锁）。
- 可买：补给券（**常驻·主来源**）+ 轮换格（信标/通用碎片/精良·优秀插件/限量星核碎片 + **星矿（改动#3：基础商品，可刷出购买）** + 稀有刷新货）+ **打捞加速券（普通/稀有/史诗三档，各减对应品质打捞 2 小时·同次可多次用·见 S10.2）**。
- **打捞加速券定价口径（数值细表约束）**：**单价偏高**（明显贵于"省下的那段时间的价值"），且**性价比低于补给券/稀缺货**，使其在前期不值得买、只在中后期星贝盈余时作为"花钱买时间"的可选项；护栏只靠"价格 + 星贝产出受控"，不设每日限量。
- 刷新：免费 + 看广告（无付费刷新）；升级给"稀有刷出/格数/购买次数/回收价"提升。

### S10.4 星空宝库（星空宝石）
- 只卖 星核（含宝库限定核）/ 专属星舰，定向兑换、一次性解锁。慢速兜底。

### S10.5 3 天行动 / 7 天扩张（不做每日任务）
- 靠正常游玩自然累积进度（不打卡）；里程碑给过程奖励、完成给宝藏。3 天=行动宝藏；7 天=扩张宝藏（第 1 次自选完整星核、之后随机三选一）。允许跨期追赶（S12）。

### S10.6 宝箱 / 背包 / 邮件 / 货币显示（含背包改动#5）
- **背包结构（改动#5）**：**「主背包」= 默认页**（补给券 / 信标 / 星核碎片 / 星空宝石 / **未开宝箱 / 礼包**）；其余为标签页：**星舰碎片 · 驾驶员碎片 · 插件 · 星核**。**点任意物品 → 弹窗（物品介绍 + 主要来源）+ 两按钮「返回 / 去使用」**；「去使用」直接跳转对应界面（如补给券 → 星港补给站）。
- 宝箱不强制立即开（先进背包）；活动结算奖励走**邮件**领取。
- **顶部货币栏**：常驻 星矿 / 星贝 / 补给券；其余货币进对应建筑时上下文加显。
