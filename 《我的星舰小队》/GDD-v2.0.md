## 0. 高概念一页（High Concept）

- **一句话**：Q 萌轻松、纯自动战斗、基地养成、星舰收集与阵容搭配的微信小游戏。
- **玩家是谁**：刚接手小破星港的"星港小队长"，在又乱又热闹的星域里修星港、救居民、收星舰、得星核，把小基地养成热闹大星港。
- **核心循环**：上线看基地变化 + 领离线/打捞 → 战前搭配（驾驶员/星舰/插件/星核 + 站位）→ 自动战斗推关 → 拿货币/碎片/插件 + 首通解锁 → 养成扩建 → 突破 Boss、推 3天行动/7天扩张 → 开宝箱得星核 → 更能打、更热闹 → 循环。
- **爽点引擎**：卡关 → 渴望更强 → 拿到牛星核 → 兵种质变·画面更爽 → 碾压推几关 → 再卡。
- **北极星**：**体验平滑不憋屈**——卡关像"今天先到这"，不像"被堵死"。
- **四层定位（本作骨架）**：星舰=打什么/底子 · 驾驶员=怎么打/打谁 · 插件=补多少数值 · 星核=发生什么质变。
- **题材基调**：Q 萌、明亮、可爱，不做压抑战争感。敌人=星盗/失控无人舰/废铁机械/星能污染舰（后者做 Boss）。
- **商业化**：首发纯广告（不卡死、不补战力），预留内购接口。
- **平台约束**：微信小游戏 / Cocos / 竖屏 / 主包 ≤4MB / 纯本地存档（预留云）。

## 5. 战斗单位模型（v2.0 修订 · 覆盖 v1.0 §5）

> v2.0 对四层模型的核心修订，**覆盖 v1.0 §5.1 的"6 武器派系"**。完整 20 星舰 + 20 驾驶员见 `GDD-附录D-星舰真源.md` / `GDD-附录D-驾驶员真源.md`，弹种见 `GDD-附录D-通用攻击视觉库.md`。

**5.1 四层定位（核心不变）**：战斗单位 = 驾驶员 + 星舰 + 插件 + 星核。星舰=动作本体（打什么）· 驾驶员=打谁/怎么打 · 插件=补数值 · 星核=规则质变。

**5.2 星舰 5 定位型（替代 6 武器派系）**：星舰按**定位**分 5 型——
- **突击舰**（刺客）：低血/高攻速/高单体爆发、收割。
- **护卫舰**（坦克）：高血高防、站前排吃伤。
- **炮击舰**（输出）：正面强火力压制。
- **支援舰**（辅助）：治疗 / 护盾 / 给友增益。
- **工程舰**（功能）：召唤单位/无人机 + 给敌方负面。
> 定位靠"属性 + 火力形态"体现，**星舰技能不指定打谁**（打谁由驾驶员定）；首发各型 4 艘、共 20。

**5.3 层级分工铁律**：**"造成/施加"归星舰，"利用/放大"归驾驶员**。
- 星舰技能 = 具体动作（开火/范围/群体/召唤/护盾/治疗/增益/**施加状态**），不写"对某类敌人增伤"。
- 驾驶能力 = 打谁/护谁（含嘲讽、守护等行为）；驾驶天赋 = 利用局面的签名规则（增伤/击杀触发/叠加/自保/放大本舰辅助），**绝不施加状态**。
- 判据：能画成一次开火/放招/召唤的=星舰；改交战规则/利用局面的=驾驶员 → 二者天然连招。

**5.4 作用形态 + 状态/控制库**：
- 形态：单体 / 小范围(十字5格) / 一片(3×3) / 一列(5格) / 一直线 / 全体 / 友方同理。**残血=血量<30%**。召唤物=同格·不移动·可被打掉·召唤源死则消失。
- 状态库：硬控=短路/沉默；软控减益=减速/虚弱/破防/致盲/易伤/燃烧/削盾/驱散；增益=加攻/加攻速/护盾/回血/增伤/净化。**平衡=硬控递减 + Boss 控制抗性**。控制由星舰施加、驾驶天赋利用。

**5.5 成长模板（三条线都有奔头 · Ron 2026-06-25 重排）**：
- **起手 Lv0（只普攻）**，Lv1 解锁第一个技能/天赋（→ 新手引导"第一场只普攻"成立）。
- **升级**：星舰每级长属性、驾驶员每级 **+1% 驾驶加成**；并 **每 20 级（Lv20/40/60/80/100）一个"升级大节点"强化**（数值/小机制，非质变）。
- **升阶/升星 = 质变线**：星舰 **A阶=大质变 / SS阶=终极质变**（C(1槽)→B(2槽)→A(3槽)→S(开星核槽)→SS、开等级上限 20/40/60/80/100、每阶全属性跳）；驾驶员 **3★=大质变 / 5★=终极质变**（能力线 +10%/星）。→ 攒碎片升 A 阶 / 升 5 星就质变变强，给足抽卡乐趣。
- **战力**：插件 + 星核 + 驾驶加成% + 升星能力% + 可量化天赋，全部计入（呼应台账"战力计入星核与插件"）。
