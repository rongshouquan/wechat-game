## 0. 高概念一页（High Concept）

- **一句话**：Q 萌轻松、纯自动战斗、基地养成、星舰收集与阵容搭配的微信小游戏。
- **玩家是谁**：刚接手小破星港的"星港小队长"，在又乱又热闹的星域里修星港、救居民、收星舰、得星核，把小基地养成热闹大星港。
- **核心循环**：上线看基地变化 + 领离线/打捞 → 战前搭配（驾驶员/星舰/插件/星核 + 站位）→ 自动战斗推关 → 拿货币/碎片/插件 + 首通解锁 → 养成扩建 → 突破 Boss、推 3天行动/7天扩张 → 开宝箱得星核 → 更能打、更热闹 → 循环。
- **爽点引擎**：卡关 → 渴望更强 → 拿到牛星核 → 兵种质变·画面更爽 → 碾压推几关 → 再卡。
- **北极星**：**体验平滑不憋屈**——卡关像"今天先到这"，不像"被堵死"。
- **四层定位（本作骨架）**：星舰=打什么/底子 · 驾驶员=怎么打/打谁 · 插件=补多少数值 · 星核=发生什么质变。
- **题材基调**：Q 萌、明亮、可爱，不做压抑战争感。敌人=星盗/失控无人舰/废铁机械/星能污染舰（后者做 Boss）。
- **商业化**：首发纯广告（不卡死、不补战力），预留内购接口。
- **平台约束**：微信小游戏 / Cocos / 竖屏 / 主包 ≤4MB / 纯本地存档（预留云）。

## S0 设计北极星
**体验平滑不憋屈。** 卡关/收手像"今天先到这"、不像"被堵死"。长时段深度靠后续内容模式（竞技场/副本），**不靠主线无限刷**。凡涉等待/卡点都要回答"如何平滑不憋屈"。

## S1 定位与核心体验
一句话：**Q萌轻松、纯自动战斗、基地养成、星舰收集与阵容搭配**的微信小游戏。玩家=接手小破星港的**星港指挥官**，修星港/救居民/收星舰/得星核/养成热闹大星港。核心体验：上线看基地变大；纯自动战斗不手操；失败回"看战斗+搭配+养成"不回操作时机；同资源不同培养路线；星核=第一稀缺品；忙几天回来不焦虑。题材 Q萌明亮。敌人 4 族群（星盗/失控无人舰/废铁机械/星能污染舰）。

## S2 目标玩家与节奏目标
四类玩家（轻/普/重/肝）。看广告不计入基础速度。**重点先保前 3-7 天体验**。
- **四档节奏目标（Ron 2026-07-02 建模确认·推翻旧"普通D21/肝帝D14毕业"结论）**：毕业=主线关卡全清（不再套死"5核"，终局养成门槛自定）。**肝档(2-3h+/天)第30天毕业 / 重度(90分钟/天)第37天 / 普通(35分钟/天，基准)第47天 / 轻度(15分钟/天)第57天**——四档顺序为肝<重<普<轻（越肝越快，但肝档不低于30天下限）。**毕业 = 进入长线循环**（深空回廊 / 极品核收集 / 拔高，见 S10.7 与 S15 路线图），不是内容终点（2026-07-02 包A）。
  - **节奏形状（参考《放置军团》实测：前期快、中期渐慢、少数大Boss长墙、破墙后加速）**：约150-190关，难度曲线"前松后紧"——首周即可清约35-40%的关卡（推图爽快），中后期进入"攒力几天、爆发式连清一批"的节奏；全程设约6个"真大Boss"关卡形成明显长墙（肝档每次≤4天、普通/轻度档在终盘大Boss处可能到8-11天，届时靠 巡逻收益 + 深空回廊/每日委托/每日推演 + 换搭配试错撑住体验——2026-07-02 包A 修订，见 S8/S10.7-S10.10）。
  - 建模工具：`game/tools/simulate-s7-progression.mjs`（`npm run simulate:s7-progression`）；当前为抽象战力单位的"形状"模型，真实资源换算/精确曲线留阶段三数值校准，届时以此形状为基准细化。

## S3 主循环（2026-07-02 包A 更新日常层）

## S4 战斗系统
- **形态**：2D 纯自动、竖屏上下；我方 3×3 上阵 5 舰、敌方 5×7；固定阵位自动交火，**只转向不位移**（技能演出=原地后坐/前倾+特效；2026-07-07 拍板③删旧"技能可短暂冲出再回位"句，与铁律统一）；入场仪式。**首发全体无限射程**（射程预留不启用），目标语义用"最近/最远/前后排"；站位影响受击优先级（默认打最前排）、目标优先级（打后排靠突袭/导弹/AoE）、左中右区域几何。
- **技能制（取消能量）**：三类异构触发（被动 / 条件 / CD型·开局即放）；靠多样性错峰、不靠统一CD避免齐爆。
- **胜负**：唯一胜利=消灭全部敌人（含召唤物）；全灭/超时 120s 判负；召唤源死即停（防软锁）。**倍速（2026-07-09 Ron 拍A·覆盖原"配 2x/3x 倍速"）**：**主线战斗全程 1×（取消倍速）**——六座墙占全关卡比例小、演出按导演节奏看全；委托/木桩/回廊/推演保留 2×/3×。**战斗暂停菜单（同批拍板）**：战斗界面右下角暂停键 → 弹「返回星港 / 重新挑战 / 继续战斗」三键；返回星港与重新挑战＝该场**视为未发生**（与下方中断口径"杀进程"同源：不耗资源/不计首通/不算失败次数/委托·回廊不耗次数额度），重新挑战＝立即原关重开；遇袭等奖励类判定按委托确定性种子、重开刷不了结果。UI 规格挂附录B 战场页（工程灰盒批落地）。

## S5 战斗单位模型（v2.0 修订 · 覆盖 v1.0 §5）

> v2.0 对四层模型的核心修订，**覆盖 v1.0 §5.1 的"6 武器派系"**。完整 20 星舰 + 20 驾驶员见 `GDD-附录D-星舰真源.md` / `GDD-附录D-驾驶员真源.md`，弹种见 `GDD-附录D-战斗视觉总谱.md` §2（原通用攻击视觉库已并入）。

**5.1 四层定位（核心不变）**：战斗单位 = 驾驶员 + 星舰 + 插件 + 星核。星舰=动作本体（打什么）· 驾驶员=打谁/怎么打 · 插件=补数值 · 星核=规则质变。

**5.2 星舰 5 定位型（替代 6 武器派系）**：星舰按**定位**分 5 型——
- **突击舰**（刺客）：低血/高攻速/高单体爆发、收割。
- **护卫舰**（坦克）：高血高防、站前排吃伤。
- **炮击舰**（输出）：正面强火力压制。
- **支援舰**（辅助）：治疗 / 护盾 / 给友增益。
- **工程舰**（功能）：召唤单位/无人机 + 给敌方负面。
> 定位靠"属性 + 火力形态"体现，**星舰技能不指定打谁**（打谁由驾驶员定）；首发各型 4 艘、共 20。

**5.3 层级分工铁律**：**"造成/施加"归星舰，"利用/放大"归驾驶员**。
- 星舰技能 = 具体动作（开火/范围/群体/召唤/护盾/治疗/增益/**施加状态**），不写"对某类敌人增伤"。
- 驾驶能力 = 打谁/护谁（含嘲讽、守护等行为）；驾驶天赋 = 利用局面的签名规则（增伤/击杀触发/叠加/自保/放大本舰辅助），**绝不施加状态**。
- 判据：能画成一次开火/放招/召唤的=星舰；改交战规则/利用局面的=驾驶员 → 二者天然连招。

**5.4 作用形态 + 状态/控制库**：
- 形态：单体 / 小范围(十字5格) / 一片(3×3) / 一列(5格) / 一直线 / 全体 / 友方同理。**残血=血量<30%**。召唤物=同格·不移动·可被打掉·召唤源死则消失。
- 状态库：硬控=短路/沉默；软控减益=减速/虚弱/破防/致盲/易伤/燃烧/削盾/驱散；增益=加攻/加攻速/护盾/回血/增伤/净化。**平衡=硬控递减 + Boss 控制抗性**。控制由星舰施加、驾驶天赋利用。

**5.5 成长模板（三条线都有奔头 · Ron 2026-06-25 重排）**：
- **起手 Lv0（只普攻）**，Lv1 解锁第一个技能/天赋（→ 新手引导"第一场只普攻"成立）。
- **升级**：星舰每级长属性、驾驶员每级 **+1% 驾驶加成**；并 **每 20 级（Lv20/40/60/80/100）一个"升级大节点"强化**（数值/小机制，非质变）。
- **升阶/升星 = 质变线**：星舰 **A阶=大质变 / SS阶=终极质变**（C(1槽)→B(2槽)→A(3槽)→S(开星核槽)→SS、开等级上限 20/40/60/80/100、每阶全属性跳）；驾驶员 **3★=大质变 / 5★=终极质变**（能力线 +10%/星）。→ 攒碎片升 A 阶 / 升 5 星就质变变强，给足抽卡乐趣。
- **战力**：插件 + 星核 + 驾驶加成% + 升星能力% + 可量化天赋，全部计入（呼应台账"战力计入星核与插件"）。

### S10.1 抽卡（星港补给站）· 三池 + **碎片化（本轮大改）**

- **同单位低阶 → 抽到高阶**：后台自动把低阶那只**分解成该单位专属碎片**（留高的；不同单位不互相分解）。

### S10.2 打捞与信标
- 获信标（三选一 / 商人 / 活动 / **深空回廊里程碑（2026-07-02 包A 新增来源，深层出稀有/史诗，见 S10.7）**）→ 选信标/打捞队/时长 → 等待 → 带回。信标档定深度、时间定数量与"稀有发现"掷骰。**打捞 = 通用碎片(舰/员)除活动外的稳定来源**（每档保底 1 种）；**也产星矿**、星贝、星核碎片/星空宝石（按档）。三档（普通/稀有/史诗）；时间档 2h/8h/24h；居民/工人/星辉货舱/各品质插件单次≤1。打捞队 1→3（打捞港升级），不会失败。**经济护栏（2026-07-06 自 v1.0 §10.2 复原·Ron 拍恢复·彻查R4）**：①**完整星核不进打捞**（保星核渠道稀缺，打捞只出星核碎片/星空宝石）；②**完整星舰＝C 阶**（同抽卡口径、不送高阶，重复转专属碎片）；③高稀有只在高档+长趟概率层、**不保底**（"单次≤1"护栏见上行）。

### S13.1 广告哲学（最高纲领 · Ron 定）
- **目标模型 = 更高留存 × 温和频次**：宁要"留 60 人每天自愿看 ~10 个"，不要"留 30 人每天看 30 个"。短期收入密度让位长线留存与口碑。玩家要**心甘情愿自己找广告看、觉得看广告很赚很爽**；禁止误触式 UI、不恶心玩家。
- **广告体验五原则（一切点位增删必须过检）**：
  1. **只锦上添花、不设过路费**：广告只出现在"收获时刻"（结算/领取/开箱/回港），绝不出现在"出发时刻"（进关前/开始玩前）。
  2. **单次会话广告决策点 ≤3、开场 ≤1**；绝不排队弹窗；"看广告翻倍"永远与"直接领取"并排可选，不为翻倍多走一步。
  3. **点位少而肥**（10 个精选）；筛选三问：自己玩烦不烦？不看亏不亏心？删掉它游戏会不会更好玩？
  4. **错过零惩罚、不做懊悔设计**：无连续签到、无倒计时压迫、不显示"你错过了多少"；超每日上限时按钮**不出现**（而非置灰）。
  5. **卡关界面零广告推送**；所有战力墙高度按"从不看广告的玩家"校准（看广告=提前破墙，不是不看多卡几天）。
- **"决策点"定义（2026-07-03 补，第 2 条计数口径）**：**只计"主动弹窗"**——系统主动跳出、需要玩家点掉才能继续的独立广告入口（如回港报告翻倍、开箱多选）；**结算画面内的并排按钮不计**（如"一键全领"旁边并排的"翻倍"键，属同一屏内的可选项，不是额外弹窗，不占决策点配额）。这样"单次会话 ≤3、开场 ≤1"才有稳定、可核对的计数标准。
- **入口视觉（Ron 定）**：正常显示、不放大、不闪动、**零红点**（凡看广告才能领的内容一律不带红点；红点只留给邮件/可升级等非广告系统）。
- 广告可带小幅战力增益（Ron 2026-07-02 放开）；**常规轨加速目标 ≈20%（上限 25%）**（2026-07-07 Ron 拍A·三档包络实测定案，取代原"≈+25-30%"；深度加速走黑市轨 S13.6），不看广告 baseline 全程成立。

### S13.3 KPI（Ron 全权委托 Claude 定稿 · 数值校准与灰度的验收标准）
- 留存：**D1 ≥40% · D7 ≥18% · D30 ≥10%**。
- 广告：**看广告人数占比 ≥70% · 观看者人均 8-12 次/天（DAU 口径 6-9 次）· 完播率 ≥90%**。
- 会话：人均 ≥2.5 次/天、单次 8-15 分钟。
- 灰度期按"广告次数 × 留存"分层验证：若重度观看者留存反而更低 = 设计有错，先修体验再谈收入；调整永远**从克制往上温和加码**（加码容易、回收难——流失的玩家不会回来）。
- **KPI 分期口径（2026-07-03 补）**：本节数字是**长期稳态验收标准**，不是"上线第一天就要命中"的达标线。分三期看：①**数值校准期**（内部、无真实玩家）——用模拟器/试玩只验证"方向不违反五原则"，不产出可比 KPI 数据；②**小流量灰度**——KPI 首次可测，允许偏离，用于快速定位体验问题并按上条"温和加码"调整；③**正式上线后**——本节数字转为持续监控的稳态基准，长期未达标才视为设计问题需回炉。

### S13.6 黑市（占位名 · 广告"类充值"轨 · 2026-07-06 Ron 拍板立项）
