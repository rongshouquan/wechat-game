## 验收要点（对 33 批四病逐条验）

1. **一眼是一家（家族脸）**：五舰共享同款胖圆糖果壳 + 大圆泡舱 —— 不看颜色也知道是一个小队/一个宇宙的。这是本批第一验收项。
2. **每舰本色鲜明、五色和谐**：蓝莓 / 天蓝 / 橙金 / 粉白 / 薰紫 各自成立，不再是"白蓝制服贴补丁"；靠家族脸统一、靠本色区分。
3. **糖果味回来**：接近基准海报的"软萌温馨"、比 33 批更甜更暖，不是精工机械玩具。
4. **五舰都像糖果飞船**：速度小艇 / 带盾胖墩 / 背炮炮艇 / 弧环奶妈 / 喷雾工程艇 —— 都是"带装备的可爱飞船"，不球化不炮管化。
5. **无星核挂件**：画面里没有棒棒糖铃铛这类突兀小物件。
6. **驾驶员到位**：站台小队长神态沉稳、构图更丰富；坐舱禁令守住（舰内无人）。
7. 顶部留白无乱字（标题后期加）；敌方/星港岛照 24/32 批标准。
