# 《我的星舰小队》插件方案 v1

状态：设计稿，待 Ron 确认。已定：插件**只分品质、不分等级**。

## 一、槽位
- 每艘星舰 3 槽：武器槽 / 技能(CD)槽 / 战术槽（固定类型，不能堆同类；同名插件一船不重复）。
- 满编 5 舰 = 15 槽。
- 槽位随**星舰升阶**解锁（已取消改装图纸）。
- **自由更换、零成本**：不分等级→无沉没成本→鼓励试搭配。这是品质制最大的好处。

## 二、品质（3 级，无等级）
| 品质 | 定位 | 效果 |
|---|---|---|
| 精良 | 量大、打底 | 单一清晰效果，数值温和 |
| 优秀 | 中期主力 | 数值更强 |
| 传奇 | 顶配追求 | 强数值 + 一条额外小效果/特色 |

- 同品质内无升级；变强靠“用更高品质替换”。
- 深度来自**效果多样性**（不是数值成长）——设计精力放在丰富词条上。例：
  - 武器槽：+伤害% / 暴击 / 攻速 / 穿透 / 对小怪或对 Boss 加伤 / 溅射。
  - 技能(CD)槽：-CD% / 开局加速首发 / 技能伤害+ / 技能范围+ / 触发概率+。
  - 战术槽：破盾值 / 减伤·护盾 / 控制时长 / 保护·嘲讽 / 回血 / 标记·易伤。

## 三、获取
| 品质 | 主要 | 次要 |
|---|---|---|
| 精良 | 关卡掉落、打捞、商人 | — |
| 优秀 | 合成（精良→优秀）、打捞(稀有)、商人 | 活动 |
| 传奇 | 合成（优秀→传奇）、行动宝库三选一 | 史诗信标打捞、Boss 首通小概率、商店稀有刷新 |

## 四、品质提升 = 合成（回答“优秀能否升传奇”：能）
- **阶梯**：精良 → 优秀 → 传奇。
- **规则**：消耗 N 个**同槽位类型**、同品质插件 → 合出 1 个**高一阶、同槽位类型**插件；具体词条在该槽位类型内随机（“定向到槽位、不定向到词条”，保留一点小赌点）。
- **起步比例（可调，细表定）**：3 同类精良 → 1 优秀；3 同类优秀 → 1 传奇（即 9 精良 → 1 传奇）。
- 作用：让“过量的精良/优秀”有去处（既是 sink、又是通往传奇的路），并给玩家**定向培养某槽位流派**的掌控感。
- 不强制合成：多余插件也可**回收换星贝**（商人）。两条出路，没有死物品。

## 五、关于“满配要 45 天”——结构比时间更重要
- 真问题不是 45 天，而是“槽位别空着”。靠量大的精良/优秀，玩家**头几天就能把 15 槽全填满**（始终满编、有功能）→ 永远不憋屈。
- 之后传奇是**渐进的品质升级**，属收尾型长期目标，不是门槛——远在“全传奇”之前就已能打。
- 而且**不需要 15 件全传奇**：关键船先上传奇收益最大。
- 合成让过量的精良/优秀**持续转化为传奇进度**，是“稳步变强”而非“干等”。
- 传奇速度是可调旋钮（掉率 + 合成比例）：嫌慢就调快；但“先填满→再升品”的结构不变。

> 北极星兑现（体验平滑不憋屈）：满编无空槽 = 不憋屈；多余插件必有去处（合成/回收）= 不浪费、平滑。
