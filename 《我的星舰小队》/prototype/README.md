# 《我的星舰小队》原型（prototype）

C1 原型的**纯 TS 逻辑核心**（战斗引擎 / 配置 / 养成），与 Cocos 表现层解耦，可在 Node 单独跑、确定性、未来可服务器复算。

## 运行（需 Node ≥ 23.6，直接跑 TS、无需构建/依赖）
- 跑战斗模拟：`npm run sim`
- 跑自动测试：`npm test`

## 进度
- **C1a-1（当前）**：战斗引擎骨架——属性 + 普攻 + 固定阵位射程目标选择 + 防御减伤 + 胜负 + 确定性 tick。
- C1a-2（待做）：效果系统（技能三类触发 / 驾驶员能力·天赋 / 插件 / 星核质变）。
- C1a-3（待做）：养成 / 掉落 / 离线 + 串成 9 关切片完整循环模拟。
- C1b（待做）：Cocos 表现层（场景/UI/色块/特效）+ 真机试玩。

## 文件
- `src/engine.ts`：战斗引擎核心。
- `src/content.ts`：最小集配置（数值取自 第二块/轻量B2-v0.1）。
- `src/sim.ts`：命令行战斗模拟器。
- `src/engine.test.ts`：引擎自动测试。
