# GDD 附录D · 通用攻击视觉库（弹种 / 特效模板）

> 目的：把"发射什么"做成**一套可复用的视觉模板**——美术只需做这十几个素材，全部星舰的普攻/技能靠"弹种 + 颜色/大小/范围"拼出来（呼应 v1.0 §4.6"通用特效模板 + 少量专属贴图"）。
> 用法：每艘星舰的攻击描述统一写成 **"发射【弹种】+ 作用(单体/小范围/一片/全体/友军) + 效果(伤害倍率/状态)"**；同类攻击**共用同一弹种描述**，不再各写一套。

| 弹种代号 | 视觉 | 典型用途 |
|---|---|---|
| **直射炮弹** | 发光实心炮弹 + 炮口闪光 | 火炮/炮击单体 |
| **聚能束弹** | 高速细长的能量束弹 | 突击/精准单体 |
| **能量飞刃** | 旋转的光刃飞镖 | 突击"近刺感"的远程化 |
| **导弹** | 小型导弹 + 尾烟（可成簇） | 范围/群伤 |
| **持续射线** | 连续不断的光束 | 激光/贯穿 |
| **电弧球** | 电球，可在敌间跳跃 | 电系/连锁/控制 |
| **光盾弹** | 飞出去的盾形光片 | 护卫单体 |
| **震荡波** | 环形冲击波/声波环 | 护卫/范围/嘲讽 |
| **治疗光弹** | 柔和绿光球 | 支援回血 |
| **护盾泡** | 蓝色护盾泡 | 支援护盾 |
| **旗光波** | 号角/战旗状光波 | 支援增益 |
| **无人机** | 小型机 + 自带激光 | 工程召唤物 |
| **扰流雾弹** | 灰紫雾团 | 工程 debuff |
| **力场环** | 束缚光环/牢笼 | 工程控制 |

**统一口径**：
- "下一次攻击 / 下一发"一律写明是**普攻**还是**技能**（如"下一次**普攻**分裂成 2 发"）。
- 范围词用通用定义：单体 / 小范围(十字5格) / 一片(3×3) / 一列 / 一直线 / 全体 / 友方同理。
- 被动若不发射弹种（如光环/护盾/形态），直接写效果即可，不用弹种。
