## 0. 设计北极星

**体验平滑不憋屈。** 首发顺应放置类天然的“被动放下 / 短时多次 / 每日回归”形态，但绝不能让玩家“撞墙、啥也干不了”：卡关或收手要像“今天先到这”，不像“被堵死”。
- 长时段、能让铁杆玩三四小时的深度，靠后续逐步增加“内容模式”（竞技场/副本等，见路线图），**不靠主线无限刷**（无限刷会摧毁“卡关→渴望→星核质变”循环并使数值通胀）。
- 凡涉及“等待/卡点”的设计，都要回答“它如何做到平滑不憋屈”。

## 1. 游戏定位与核心体验

一句话：**Q 萌轻松、纯自动战斗、基地养成、星舰收集与阵容搭配**的微信小游戏。

玩家是刚接手小破星港的“星港小队长”。星域因星路断裂、无人舰失控、星盗捣乱、废铁机械乱飞而又乱又热闹。玩家修复星港、救回居民、收集星舰、获得星核，把小基地养成热闹大星港。

核心体验：
- 上线看到基地变大、居民更多、飞船更多。
- 战斗完全自动，玩家不手动放技能。
- 失败后的应对回到“看战斗 + 战前搭配 + 养成选择”，不回到操作时机。
- 同样资源，不同玩家有不同培养路线。
- 星核是第一稀缺品（=本作的“宝物”），拿到能明显改变战斗体验。
- 忙几天回来不焦虑，仍能继续收资源、派打捞、推主线。

题材基调：Q 萌、明亮、可爱，不做压抑战争感。
敌人族群：捣蛋星盗 / 失控无人舰 / 废铁机械团 / 星能污染舰（后者造型夸张，适合做 Boss）。
主线目标：修复星港 → 救回居民 → 打通星路 → 收集星舰与星核 → 养成热闹大星港。

## 2. 目标玩家与节奏目标

四类玩家（默认目标玩家基本每天有意愿上线，但不做每日压力，忙 1-2 天回来不断档）：
- 轻度：每天约 10-20 分钟；目标第 28 天主线毕业或接近，形成 3-5 核编队。
- 普通：每天约 25-45 分钟；**主要平衡对象**；目标第 21 天主线毕业或接近，接近 5 核编队。
- 重度：每天约 60-120 分钟；目标第 14 天挑战最终 Boss/完成主线，第 21 天形成 5 核编队。
- 肝帝：每天 2-4 小时+；只用于压力测试；目标第 7 天进入最终星域深处、第 14 天主线毕业。

> 以上为**目标节奏（定性）**，第二块节奏表负责定量验证；看广告不计入基础成长速度，须单独验证“不看广告版本”。
> 平衡口径：**普通玩家=平衡基准、轻度=体验下限、肝帝=压力上限（不作调参基础）、重度居中**。首发/原型期**重点先保证前 3-7 天体验**（留存关键窗口）；21/28 天长曲线是目标，靠节奏表 + 上线后真实数据持续调，不是承诺。

## 3. 主循环

```
上线看基地变化 → 领取离线积累 + 打捞结果
→ 分配资源、调整 驾驶员/星舰/插件/星核
→ 自动战斗推关
→ 获得软货币、信标、插件、碎片等
→ 扩建基地、强化小队
→ 突破 Boss、推进 3 天行动 / 7 天扩张
→ 获得星辉货舱、行动宝藏、扩张宝藏（星核）
```
所有系统都服务两个结果：① 小队更能打；② 基地更热闹、更大、更有成就感。

## 4. 战斗系统

### 4.1 基本形态
- 2D **纯自动战斗**，玩家不手动放技能、无舰长过载。策略全发生在战前（选驾驶员/星舰、装插件/星核、调站位、定培养路线）。
- 竖屏**上下布局**：我方在下，敌方在上。
- 我方 **3×3 九宫格摆阵、上阵 5 艘星舰**，只在战前调整，战斗中不手动移动。
- 敌方视觉上无固定数量/阵型，底层用 **5 行 × 7 列（可配置）** 格子承载小怪/精英/多格 Boss/炮台/召唤物（由 3×7 加深为 5×7：保 7 列宽度、行数加深，让敌方战场更接近方形、摆怪更灵活/随机；**做成配置参数，最终行列比 C1b 真机布局时定**）。
- **固定阵位自动交火**：双方大部分时间待在各自阵位坐标；技能可短暂冲出阵位（跃迁/穿场/撞击/后撤）再回位。小幅漂浮/受击抖动/后坐力/闪避用程序位移实现，不新增美术。
- 入场仪式：开战后 5 舰从屏幕下方平行飞入 → 移动到各自阵位 → 正式开战（情绪价值，不影响数值，复用星舰图像程序实现）。
- **站位的战斗作用（我方 3×3 vs 敌方 5×7·可配置）**：敌方格子由 3×7 加深为 **5×7**（保 7 列、行数加深，更接近方形、摆怪更灵活；可配置，最终 C1b 定）。**首发默认全体无限射程**（射程作为机制预留、首发不启用）。**目标语义用"最近/最远、第一排/最后一排"**（与具体行数无关；引擎 frontmost/backmost 即"第一/最后排"）。站位影响：
  - **受击优先级（纵深）**：默认攻击对方**最前排**——前排先挨打 / 先被集火；后排相对安全（被晚打），但敌方可有“后排点名 / 穿透”打你后排（**后排 ≠ 绝对安全**）。
  - **目标优先级 = “打后排”的克制载体**：想先点掉敌方后排炮台，靠**突袭手能力（改目标优先级到后排）或导弹 / AoE**——从“够不着”的硬锁改成“被默认忽略、需主动优先”的软克制，更平滑不憋屈。
  - **左 / 中 / 右（区域）**：范围技形状、打一列、点名某区等几何（3 列不强对 7 列，按区域判定）。
  - 摆阵还影响支援 / 保护范围、部分关卡机制。**前排不只放坦克**；只在战前调整。

### 4.2 技能制（取消能量）
- **取消能量条**；术语用“技能”（不用“大招”，避免“个个威力巨大、开场齐爆”的暗示）。
- 技能**三类异构触发**（靠多样性天然错开节奏、避免 5 舰齐爆）：
  - 被动型：常驻（如强化普攻、光环）。
  - 条件型：满足条件触发（如击杀 N 个后、某方阵亡到数、己方血量低于 X%）。
  - CD 型：每隔 CD 自动释放，**默认开局即放第一次**。
- 避免齐爆的关键在“把星舰技能与驾驶员效果设计好”，不靠统一 CD 硬错峰。

### 4.3 胜负与时长
- **唯一胜利条件 = 消灭全部敌人（含召唤物）**；我方全灭 / 时间耗尽 / 其它 → 一律判负。
- 召唤约束：召唤不能无限，否则永远清不完 = 卡死。规则 = **击杀召唤源（Boss / 召唤单位）后停止召唤**，再清残余；配合 120s 超时兜底，不会软锁。
- 关卡超时 = **120 秒** 判负（失败提示如“超时未击破”）。
- 时长目标（1x，配合 2x/3x 倍速压缩）：战力匹配正常推进 ~20-30s；战力碾压（越级/回刷）开场一波 ~5-10s 秒清；卡关打到 120s 超时判负；Boss ~45-90s，含 2-3 阶段。

### 4.4 战前 / 编队界面（放置军团式）
- 战前界面即战场界面：上方显示敌方阵容，下方显示九宫格轮廓，**默认延续上一把阵容与站位**。
- 点击已上场星舰 → 弹窗列出玩家拥有的全部星舰 → 可替换 / 可只下场；上阵 <5 时点空位可上阵，已满 5 点空格无反应。
- 弹窗内有按钮进入**单艘星舰备战界面**：调整该船的**驾驶员 / 插件 / 星核**。
- **每艘上阵星舰必须配一名驾驶员**（战斗单位 = 驾驶员 + 星舰，空船不能上阵）；驾驶员数 < 上阵位则上不满——故“救 / 抽驾驶员”是扩充编队的关键（呼应新手流程救人才能多上一艘）。
- 每艘星舰各自记忆自己的 loadout（换上换下不丢配置）。
- **实例制**：驾驶员 / 插件 / 星核均为实例，同一实例同时只能装在一艘船上；在备战界面挑选时，已装在别船的会在图标旁标“所在星舰”，仍可直接点选装到当前船——系统自动从原船卸下并提示。
- 战前可预览当前关卡**基础敌情**（敌人类型 / 数量 / 主要位置 / 是否有 Boss）；**所有关卡都不剧透关键威胁**（护盾 / 后排炮台 / 召唤 / 狂暴 / 点名等），让玩家进战斗自己观察、再回来调整搭配（呼应 §4.5）。

### 4.5 表现原则与克制
- “平时清楚，关键时刻震撼”：普攻不乱闪；技能爆发、破盾、连锁击杀、星核触发、Boss 阶段转换时才出强表现。不做满屏常驻乱炸。
- **克制存在，但首发不锁死克制类型**（星舰与敌人未设计完，避免提前定死）。设计意图：过新手期后，卡关靠玩家**观察战斗 → 调整阵容/阵型站位**取胜，实在不行才等资源提升战力。
- **知识传递路径**（不靠复杂失败分析页）：战前敌情预览 + 新手关逐类教学每种威胁 + 失败提示点名威胁类型。
- 失败界面轻量：显示“战斗失败”+ 一句胜负提示 + 单位输出统计；两个按钮“再次挑战 / 返回星港”。不做复杂失败分析、不做提升路径建议。

### 4.6 实现架构（配置驱动 + 效果系统）——写代码时遵守
应对“驾驶员 × 星舰 × 插件 × 星核”的海量组合，代码必须做成“积木 + 引擎”，而非逐组合硬编码：
- **一切皆配置**：每艘星舰/驾驶员/插件/星核都是一份数据，只描述“拥有哪些效果(effect)”，不把组合写死在代码里。
- **效果 = 一套有限、可复用的积木**：
  - 修正类（+X% 伤害 / -Y% CD / +护盾…）—— 插件、驾驶天赋数值用它。
  - 触发类（开局 / 每隔 N 秒 / 击杀时 / 血量低于 X% / 受击时…）—— 技能三类触发用它。
  - 行为类（选目标=最近/后排/最脆/最肥/护盾…）—— 驾驶能力用它。
  - 动作类（伤害 / 范围爆炸 / 召唤 / 治疗 / 施加状态…）—— 技能、星核效果用它。
- **战斗引擎**：不关心“是什么组合”，只把一艘船所装 驾驶员+星舰+插件+星核 的全部 effects 收集起来，逐 tick 按规则结算。千变万化的战场表现是“涌现”出来的，不是逐组合写死。
- **复杂度只跟“积木种类”线性相关，不跟“组合数”相关**：加一个新星核 = 加一条配置 +（顶多）一个新积木，不动引擎、不影响其它组合。
- 引擎 = **Cocos Creator 3.8.x（TypeScript）**。
- **【开发硬原则·逻辑与引擎解耦】所有核心逻辑（战斗 / 配置 / 数值 / 存档 / 推荐目标等）必须写成普通 TS 模块，不依赖 `cc` 节点 / 组件 / 场景 / 装饰器，必须能在 Node/Vitest 里单独测试。** Cocos 只负责画面显示、点击输入、资源读取、生命周期调用；**反向约束：表现层不写游戏规则**，只读逻辑状态来渲染、把输入转交逻辑。好处：可单测、数值秒级迭代、未来可服务器复算、开发不被编辑器卡住（`prototype/` 即此原则的活样板）。用大量自动化测试覆盖效果叠加的边界交互。
- 视觉“千变万化”靠**通用特效模板（弹道/爆炸/拖尾 + 颜色/路径/震动）+ 少量专属贴图**程序化拼（见 §8），逻辑组合便宜、视觉差异靠复用。

## 5. 战斗单位：四层定位（核心）

一个战斗单位 = **驾驶员 + 星舰 + 插件 + 星核**。四层职责彻底分开、互不重叠：

| 层 | 拥有 | 升级/升阶给什么 | 边界（不碰什么） | 一句话 |
|---|---|---|---|---|
| **星舰** | 基础属性、技能（自带战斗动作：普攻/被动/条件技/CD技）、3 插件槽+1 星核槽 | 升级→属性（+里程碑解锁基础技能）；升阶→开槽 + 解锁/强化高级技能 | 不决定目标/行为、不质变 | 打什么、底子多强 |
| **驾驶员** | 驾驶能力（目标选择/行为 AI）+ 驾驶天赋（独特被动） | 升级→强化驾驶天赋数值；升星→解锁/深化驾驶能力 + 天赋质变（**无原始属性**） | 不自带独立技能（大招归星舰），只改写/增幅 | 怎么打、打谁 |
| **插件** | 定向数值/小效果（品质制） | 不分等级；换品质 / 合成升品 | 只微调数值，不解锁技能/不改行为/不质变 | 某方向补多少数值 |
| **星核** | 规则级质变 | 获取即质变（一船一核） | 最稀有、最夸张、最视觉 | 发生什么质变 |

**关键区分（星舰技能 vs 驾驶员能力）**：星舰技能是“动作本体”（自带、独立触发）；驾驶员能力是“怎么用这些动作、打谁、打出什么味道”（改写目标、增幅打法），**驾驶员绝不独立放一个技能**。
**例（导弹舰）**：星舰给“齐射”动作 → 突袭手能力让齐射“专锁后排” → 武器插件给“暴击+溅射”数值 → 小太阳星核把“齐射”变成“吸附爆炸的小太阳”。四层各管一段。

### 5.1 星舰
战斗主体与主要收集追求。星舰有三个分类轴：
- **武器派系（打什么·火力硬件）**：火炮型（直射稳定，自由型多为此）/ 导弹型（弹道范围爆炸）/ 激光·射线型（穿透·破盾）/ 无人机型（召唤多目标）/ 壁垒型（高耐久 + 防御投射）/ 电磁·脉冲型（削弱·破盾·短路）。决定火力形态、技能本体与视觉。
- **功能定位**：自由型（百搭，武器多为火炮型，后期也能用）/ 流派型（围绕某武器派系深度设计，配契合角色最强）/ 专属型（强绑某驾驶员，“准第一稀缺”，长期追求）。
- **阶级（养成进度，非独立稀有度）**：C / B / A / S / SS，靠**升阶**（碎片）提升（见 §6）。本作**不另设抽卡稀有度**——本体差异在“种类（派系 / 定位 / 技能）”，强弱看你升到几阶。
- **专属型绑定**（放置军团专属宝物式）：**谁都能开，但其绑定的驾驶员来开会发挥专属效果**。

能力：普攻（持续输出）、技能（三类触发）、被动（武器派系倾向）。

### 5.2 驾驶员
角色（数量见 §14）。不是战斗主形象（星舰才是），决定同一艘船在不同人手里打出不同味道。**驾驶员不提供原始属性（不加血/攻），只贡献两样东西：**
- **驾驶能力（目标选择 / 行为 AI）**：决定本舰打谁、怎么与站位和敌方机制互动（如锁后排最脆、拉嘲讽护后排、优先打护盾敌人）。行为离散，在升星里程碑分阶解锁更强版本。
- **驾驶天赋（改变“怎么打”的独特被动）**：本舰的签名机制（如越打越热、斩首、维护、附带短路）。数值随升级平滑成长，升星时质变跃升。

驾驶员按**战术角色**分（与星舰“武器派系”正交的另一个轴）：炮手·输出 / 突袭手·刺客（打后排脆肥）/ 护盾手·坦克（抗线保护）/ 工程师·辅助（奶·加速·支援）/ 干扰手·控制（破盾·短路·削弱）/（舰长·指挥：全队光环，首发可精简）。

> **星舰决定“火力形态”，驾驶员决定“这火力怎么用、打谁”——二者正交相乘，是玩家应对不同战况的克制工具箱**（高护盾→电磁/激光+干扰手；后排点名→导弹+突袭手；敌方爆发→壁垒+护盾手；小怪海→导弹/无人机+炮手；续航→工程师）。

**阶级（养成进度，非独立稀有度）**：星级 1★–5★，靠**升星**（碎片）提升（见 §6）。本作不另设抽卡稀有度——驾驶员差异在“角色 / 天赋种类”，强弱看你升到几星；高星 = 潜力上限更高，但不无脑碾压，低星在特定搭配下长期有用。

### 5.3 插件
替代传统多部位装备。每艘星舰 3 槽：**武器槽 / 技能(CD)槽 / 战术槽**（固定类型、不能堆同类、同名一船不重复）。槽位随星舰升阶解锁。
- **只分品质、不分等级**：精良 / 优秀 / 传奇。同品质内无升级；变强靠“用更高品质替换”。深度来自**效果多样性**而非数值成长。
- **自由更换、零成本**（无沉没成本，鼓励试搭配——这是品质制最大的好处）。
- **品质提升 = 合成**：阶梯 精良→优秀→传奇；规则“消耗 N 个同槽位类型同品质 → 1 个高一阶同类，词条在该槽位类型内随机”；起步比例 3 精良→1 优秀、3 优秀→1 传奇（可调）。多余插件也可回收换星贝，无死物品。
- 满配心态：靠量大的精良/优秀，玩家头几天就把 15 槽填满（始终满编、不憋屈）；传奇是渐进的品质升级、收尾型长期目标，不需 15 件全传奇。

### 5.4 星核（第一稀缺品 = 本作“宝物”）
一船最多 1 核，同名核一场只 1 个。负责**稀有质变**（强规则变化/强组合/强稀缺/战斗演出），不要求每个都夸张演出。
设计原则：大部分看起来就厉害；不只是普通属性加成；不做废核/陷阱核；可有通用型与场景型（场景型在对应场景必须很强）。
- **稀缺性落在“拿到指定的 / 最强的那个”，而非“获得星核本身”**——获得本身要有一定流速做中期爽点引擎，否则“渴望”转不动变干等。卡关墙的长短是数值活，长到制造渴望、短到不流失。
- 玩家短期爽点闭环：卡关 → 渴望 → 获得牛核 → 兵种质变、视觉更爽 → 碾压推几关 → 再卡 → 再渴望。
- 三条获取渠道并行（随机为主、定向兜底）：
  1. **扩张宝藏**：每 7 天随机三选一（第一次自选）。
  2. **星核碎片 → 随机合成**：碎片攒够随机合成 1 颗（赌、惊喜）。
  3. **星空宝石 → 星空宝库定向兑换**：攒够宝石指定兑换想要的核（慢速兜底）。
- **里程碑首核（固定·新手核·过载核心）**：第一个 Boss 首杀掉落**固定的“过载核心”**（人人相同、**非三选一**）。效果：**把本舰普攻改成“原子炮”——开局即放、一发对以目标列为中心约半个敌方战场的范围造成巨量伤害，但普攻间隔变 10 秒**。前期拿到超爽（开局一炮秒一大片、视觉突出），中期够用，后期过渡、**非毕业核**，让新人第一次就强烈感到“核 = 变强很多”。（成就门控，正常玩家约第 3 天。）
- **宝库限定核**：2-3 个最强/最酷的核只能用星空宝石在宝库兑换、随机开不出。护栏：限定核可强可酷，但不做“通关必需”、不做付费墙。
- 星核不进常驻抽卡池；首发不开星核实验室、不做重复星核深层养成（留 P1）。星核装配由星舰升阶解锁星核槽后完成。
- 星核例子（效果第二块细化）：小太阳 / 黑洞 / 星鲸 / 银河烟花 / 时光糖 / 超级护罩 / 幸运扭蛋 / 舰队合唱 / 彩虹轨道 / 星门跃迁。

### 5.5 战斗词条（哪些属性、由谁提供）
属性清单现在定（框架），具体数值 / 成长曲线留第二块。
- **基础属性（星舰升级提供，原始数值）**：血量、攻击、攻速、防御 / 减伤。
- **定向词条（插件提供，可自由换 = 可调方向）**：暴击率、暴击伤害、破盾（对护盾效率）、技能急速 / 缩 CD / 首发加速、治疗强度、控制时长 / 抗性、对小怪或对 Boss 加伤等。
- **独特机制（驾驶天赋提供，非通用词条）**：如越打越热、斩首、维护——是“机制”，不是一条通用属性。
- **规则质变（星核）**：改写技能本身，不在词条体系内。
> 原则：基础属性走星舰、定向数值走插件、独特机制走驾驶员、质变走星核——与四层定位一致，不重叠。

## 6. 养成与升阶

- **星舰**：升级（消耗星舰合金）→ 基础属性 **+ 早期等级里程碑解锁基础技能**；升阶（星舰碎片 + 通用星舰碎片）→ 阶段性开槽（插件槽 1/2/3、星核槽）+ 解锁 / 强化更高级技能。
- **驾驶员**：升级（消耗驾驶记录）→ **强化驾驶天赋数值**（无原始属性）；升星（驾驶员碎片 + 通用驾驶员碎片）→ **解锁/深化驾驶能力 + 驾驶天赋质变**。
- **插件**：不升级，换品质 / 合成升品（见 5.3）。
- **星核**：获取即用，不常规养成。
- 升阶材料说明：**专属碎片**来自抽卡重复（每单位）；**通用碎片**（舰/员，万能投放）来自活动，让稀有单位也能升阶。
- UX（北极星兑现）：升级一键升到当前资源可承受上限、并显示战力增量，不让玩家一级一级戳。

> **战力数值对玩家展示**：把 驾驶员+星舰+插件+星核 的数值层折成统一战力值，玩家据此判断“这关打不打得过”。升级 / 升星 / 升阶都抬高可量化部分，**任何养成都会让战力上涨（含驾驶员升星）**；只有“目标 / 行为”与“规则质变”这类不可量化价值不计入战力，故“战力够也可能没赢”——这是健康的（避免纯战力碾压）。

## 7. 基地养成

基地是主界面，也是长期成就感核心。第一天能看到很多未来空位，功能区分阶段开放。

**主界面入口（UI 按钮，不是建筑）**：
- “出战”按钮（屏幕底部中 / 右）——进入主线推关。
- 3 天行动 / 7 天扩张入口（屏幕左侧 UI 按钮）。
- 取消“指挥塔”建筑。

**建筑列表**：
- 船坞：**星舰管理中枢**——查看全部星舰，对每艘进行升级 / 升阶培养与装配（驾驶员 / 插件 / 星核）。（战前编队界面是它的“出战前”精简版，二者共用单舰备战界面。）
- 居住舱：居民数量、离线储存上限。
- 打捞港：派打捞队；**升级解锁更多打捞队（1→3）**（原“星港”的增打捞队功能并入此处，取消星港建筑）。
- 商人小站：星贝商店（回收 + 购买 + 稀有刷新，见 §10.3）。
- 星港补给站：抽卡（招募 / 整备，见 §10.1）。
- 研究塔：全队小加成 / 基地科技。
- 星核展厅：展示已获星核、提供轻量全队加成与收藏成就感（**普通建筑，不再用“奇迹建筑”概念**）。

解锁节奏：D1 船坞 / 居住舱 / 星港补给站 → D2-3 打捞港 → D3 后商人小站（首次出现多余物品回收需求时）→ D4-7 研究塔 → D7 后星核展厅（拿到首核后）。
建筑规则：蓝图获得即激活 1 级；升级消耗星矿 + 少量指定材料、**立即完成无等待**；1-10 级（1 级解锁功能，2-5 提升数值/外观，6-10 解锁更明显外观与高级加成）。

**基地人口（精简为两类，其余效果并入对应建筑升级）**：
- 居民：提高离线储存上限，让基地更热闹。
- 工人：形成工程力，降低基地升级消耗，解锁施工外观变化。

**离线产出（挂机收入骨架，方案2弱化版）**：
- 基地被动产出软货币（星矿/星舰合金/驾驶记录），离线累积，上限由居住舱决定。
- 产出速率 = 建筑等级基础 × 星域系数，**随“已通关最高星域”提升**——推主线既给一次性首通奖励、又永久抬高离线基线。
- 上线一次性“领取离线积累”，保留“攒了一大笔”的领取仪式感。
- 重复挑战关卡不刷资源（不破坏）；不做“挂机刷当前关”的战斗画面。

## 8. 关卡

首发星域与关卡规模见 §14（约 3-4 星域、50-70 主线节点、8-10 精英关、3-4 Boss；最终由原型反推）。
- 节点复用：普通关用 8-10 套模板复用；精英关用约 8 类机制做变体；每个星域 Boss 各有主题与 2-3 阶段。
- 分工与插件掉落阶梯：
  - 普通关=爽推 + 软货币；首通**概率掉精良插件**（前期概率高/接近保底，快速填满槽位）；小概率普通 / 稀有信标。
  - 精英关=机制考验 + **必得稀有信标**；首通**概率掉优秀插件**；小概率史诗信标。
  - Boss 关=阶段战 + **必得史诗信标** + 补给券 + 首通**优秀插件（小概率传奇）** + 里程碑（含首个 Boss 的星核宝藏）。
- 敌人配置：底层 **5 行 ×7 列（可配置，首发暂定，最终 C1b 真机定）**；普通关常规 6-12 小怪**一次性刷新出场**（不做分批进场——分批刷怪留给未来理想版的塔防玩法）；精英关 2-5 精英 + 少量小怪；Boss 关单 Boss 或 Boss+小怪 / 炮台 / 召唤；保留位置概念以做打一列 / 打后排 / 后排点名 / 中路轰炸等机制；敌人逻辑位置默认固定，仅视觉小幅位移后回位。
- **敌人设计（post-freeze 补充）**：敌人与我方**共用引擎**（配置 + 效果积木）——一个敌人 = 属性 + 站位 + 行为(目标选择) + 机制 effect，**同样固定阵位、不移动**（首发无限射程；“打后排”靠突袭手的目标优先级或导弹 / AoE，不靠射程也不靠移动）。**核心原则：敌人"威胁类型"必须对应玩家"克制工具箱"**（护盾↔破盾、后排↔打后排、小怪海↔清群、爆发↔扛/护罩、召唤↔清场），否则换搭配无意义。族群见 §1，机制见上；原型具体 4 类敌人见 第二块/轻量B2。
- 重复挑战：重在体验与冲记录，不是刷资源（普通/精英/Boss 重复奖励都明显低于首通）。
- 第 7 天口径：不是打穿完整主线，而是完成第一阶段大目标、重度/肝帝进入最终星域；第 7 天星核是打开后半段体验的钥匙。

> 战斗结算需为**确定性 + 配置/种子驱动、可单独跑**（为未来竞技场服务器复算预留），首发不接真实联网。

## 9. 资源与经济

### 9.1 货币清单
- 软 / 流通货币 ×4：星矿（建基地，**溢出可回收为星贝**）/ 星舰合金（升星舰）/ 驾驶记录（升驾驶员）/ **星贝（基础流通货币，在商人小站消费）**。前三者由主线每战 + 基地离线稳定产出；**星贝 = 主线稳定给 + 打捞 + 回收（不走离线产出）**。
- 抽卡券 ×1：补给券（单一券 + 抽取选池）。
- 升阶材料：专属碎片（每单位进度条，抽卡重复）+ 通用星舰碎片 / 通用驾驶员碎片（活动产出、万能投放）。
- 星核体系 ×2：星核碎片（随机合成）/ 星空宝石（定向兑换）。
- 信标 ×3：普通 / 稀有 / 史诗。
- 物品：插件（精良/优秀/传奇）、本体（驾驶员/星舰）、**星核 / 专属星舰**（终产物，装备 + 收藏）。
- 基地人口：居民 / 工人。
- 奖励包装（非货币，先进背包再开）：星辉货舱 / 行动宝藏 / 扩张宝藏。
- **已砍**：太空补给、改装图纸、星辉信标、奇迹信标、招募券/整备券（并入单一补给券）。

### 9.2 每玩法的“主要产出（玩它的理由）”
- 主线推关：软货币大头（含星贝）+ 首通解锁（救回驾驶员本体/建筑/系统）+ 补给券 + 精良插件 + 居民 + 抬高离线基线 + 首颗星核（Boss 星核宝藏）。
- 离线产出：软货币稳定骨架。
- 基地建设：永久全队加成、离线上限、容量（SINK + 乘区）。
- 打捞：稀有物滴流（碎片/插件/星核碎片/星空宝石，按信标档）。
- 抽卡 + 赞助补给：驾驶员/星舰本体（收集与编队广度）。
- 商人小站：中端定向货 + item sink + 逛店惊喜。
- 星空宝库：定向兑换 星核（含限定核）/ 专属星舰。
- 3 天行动：插件（传奇）+ 定向通用碎片 + **星辉货舱**（完成奖励，开出星核碎片 / 星空宝石 / 信标）。
- 7 天扩张：星核 + 居民/工人 + 史诗信标。

### 9.3 每资源的主要 / 次要渠道（摘要）
- 软货币：离线 + 主线每战（主）；活动、打捞普通信标（次）。
- 补给券：主线/Boss 首通 + 3天/7天活动 + 赞助补给（主）；商人、打捞小概率（次）。
- 本体：抽卡（主）；主线救援/里程碑（次）。
- 通用碎片：3天/7天活动（主）；商人（次）。
- 星核碎片：打捞(稀有/史诗) + 精英/Boss（主）；商人/活动/星辉货舱（次）。
- 星空宝石：打捞(史诗) + 7天完成（主）；星辉货舱/商人/精英 Boss 小概率（次）。
- 星核：扩张宝藏随机三选一 + 碎片随机合成 + 星空宝库定向兑换（里程碑：首个 Boss 星核宝藏、扩张宝藏第 1 次自选）。
- 专属星舰：星空宝库定向兑换。
- 信标：普通=普通关小概率+商人；稀有=精英必得+7天过程；史诗=Boss必得+7天完成。
- 插件：精良=关卡/打捞/商人；优秀=合成+打捞/商人；传奇=合成+行动宝藏+史诗打捞/Boss小概率/商店稀有刷新。
- 居民：主线推进救回 + 7天过程；工人：7天扩张 + 基地里程碑。
- 星贝：主线稳定给（主）+ 打捞 + 回收（多余插件 / 信标 / 溢出星矿）。

> 星核碎片/星空宝石来源较广是**有意为之**（每个活动都有星核进度=平滑不憋屈）；总量靠“主次加权 + 合成成本/兑换价”控制，不靠砍来源。

## 10. 各玩法系统

### 10.1 抽卡（星港补给站）
- 主动抽取系统，消耗**补给券**，抽取时选池：招募池（驾驶员）/ 整备池（星舰）。**两池只出本体**，重复转对应专属碎片；不出插件、星核不进池。
- **保底 = 阶级地板**：N 抽必出随机一个 ≥3 星驾驶员 / ≥A 级星舰（等于一次送够碎片、直接到中阶，免去从 1 阶慢慢肝）；**若该本体已拥有，自动折成等量碎片**（投给它或转通用），无重复尴尬。顶级（5★ / SS）另设更长保底兜底（数值第二块）。
- **补给券脉冲节奏**：前期快速凑齐大部分驾驶员/星舰（且前期升阶不重耗碎片）；够用后可屯券（**券不过期**），出新角色/要升阶时再针对性抽。“抽→够用→屯→要用了再抽”，不制造每日压力。
- **赞助补给**：看广告得补给券（首发纯广告核心变现点之一，每日上限）。

### 10.2 打捞与信标
- 流程：获得信标 → 选信标 → 选打捞队 → 选打捞时间 → 等待 → 带回奖励。信标决定奖励池，时间决定数量与稀有追加。
- **时间收益递减**：时长越长总收益越多、但每小时效率递减（示意：2h≈100、4h≈180、8h≈320 星矿）。鼓励活跃玩家多跑短趟，休闲玩家一趟长打省心。
- 信标 3 级（加法分层）：普通（软货币 + 星贝）→ 稀有（+星核碎片、通用碎片、插件）→ 史诗（+星空宝石）。
- 打捞队 1→3（靠**打捞港升级**解锁）；**打捞不会失败**；不做复杂打捞小游戏。
- 广告加速：看广告缩短打捞剩余时间，**每日总次数上限 3 次（打捞港升级可提至 5 次）**。减量方式**按档位固定减**（短档减得少、长档减得多，如 2h 档减 30min、8h 档减 2h、24h 档减 6h），不用“统一减 1 小时”（对长档几乎无感）。具体值第二块定。

### 10.3 商人小站（星贝）
- 定位：item sink + 中端定向购买 + 逛店惊喜。
- 可卖（回收→星贝，有折损）：多余 / 重复插件（各品质）、各档信标（普通 / 稀有 / 史诗都可回收，但稀有 / 史诗价值高、一般不建议卖）、**溢出的星矿**（建筑封顶后星矿易溢出，回收为星贝是它的去处）。本作无专门“杂项垃圾”类别。**不可回收**：本体、星核、专属星舰、各类碎片、星核碎片、星空宝石、补给券、星舰合金 / 驾驶记录（留着花）。
- 可买（固定格 + 周期刷新）：信标、通用碎片、精良/优秀插件、少量补给券、限量星核碎片。
- **稀有刷新**：少量随机格低概率刷出贵的好东西（传奇插件 / 史诗信标 / 较多碎片 / 极低概率少量星空宝石），标价高、限购 1 次、刷到才有；养“商人”提高稀有刷出率与质量。
- 升级效果：回收得更多星贝、商品格增加、高品出现率↑、稀有刷新质量↑、可手动刷新次数↑。
- **每日 1 次看广告刷新商店**（也增加刷到稀有商品的机会，是个良性广告点）。
- 防套利：回收折损、周期购买上限、刷新上限、不可回收表、星贝不能无限循环刷资源。

### 10.4 星空宝库（星空宝石）
- 只卖**星核（含宝库限定核）/ 专属星舰**，定向兑换、一次性解锁（买过显示已拥有）。慢速兜底渠道。

### 10.5 3 天行动 / 7 天扩张（不做每日任务）
- **进度怎么推**：靠玩家**正常游玩自然累积**（推关、升级、打捞、抽卡等日常行为计入进度），不是一串“必须专门做”的打卡任务——不制造每日压力（呼应“不做每日任务”）。设几个里程碑节点（进度到段位给过程奖励），完成给完成奖励、结算给宝藏。具体目标项与数值第二块细表定。
- 3 天行动（短周期）：过程奖励偏**信标 / 补给券 / 通用碎片**（少给每战已有的软货币）；完成奖励=星辉货舱；结算奖励=**行动宝藏（传奇插件 / 指定通用星舰碎片 / 指定通用驾驶员碎片 三选一）**。
- 7 天扩张（中周期）：过程=更高量基础资源 / 居民 / 工人 / 稀有信标 / 通用碎片；完成=史诗信标 / 星核碎片；结算=**扩张宝藏（第 1 次自选完整星核；第 2 次起随机星核三选一；留广告接口、本轮不做核广告收益）【机制不改，仅更名】**。
- 均允许**跨期追赶**（见第 12 节）。

### 10.6 宝箱、背包、邮件与货币显示
- **宝箱拿到不强制立即开**：星辉货舱 / 行动宝藏 / 扩张宝藏先进**背包**，玩家想开时再去背包打开。
- 星辉货舱：随机量 星核碎片 + 星空宝石 + 3~5 个随机品质信标；开箱展示 3 选项免费选 1，看 1 次广告再选第 2 个。
- 行动宝藏 / 扩张宝藏：内容见 10.5。
- **邮件系统**：行动宝藏 / 扩张宝藏等活动结算奖励、系统补偿等通过**邮件**发放，玩家点邮件领取到背包。
- **顶部货币栏（上下文显示）**：主界面只常驻显示 **星矿 / 星贝 / 补给券**（最常用）；其余货币只在进入对应建筑时上下文加显（如进补给站显示补给券、进船坞显示合金 / 驾驶记录 / 碎片、进打捞港显示信标）。
- **背包系统**：除软 / 流通货币（星矿 / 星舰合金 / 驾驶记录 / 星贝）外，**其它资源都在背包可查看**——补给券、信标、通用碎片、各单位专属碎片、星核碎片、星空宝石，以及未开宝箱、未装插件、可用礼包。背包让“先拿着、想用再用”成立（攒着的宝箱不过期，配合跨期追赶）。

## 11. 新手引导（分段）

- **首日 / 前 10 关强引导（10 步；跨度超过 10 分钟）**：
  1. 进入小破星港，获得第一个驾驶员 + 第一艘自由型（火炮型）星舰。
  2. 第一场自动战斗：**只有普通攻击、没有技能**（先建立“自动打”的基线）。
  3. 胜利 → 获得星矿 / 星舰合金（看到资源进账）。
  4. 回船坞 → **升 1 级星舰（花星舰合金）→ 解锁第一个技能**。
  5. 第二场战斗展示刚解锁的技能，明显更快（感受“变强了”）。
  6. 第二场结束掉落第一个精良插件 → 引导进船坞 / 备战界面装入武器槽 → 下一场感受插件效果（教“装插件”，并告知插件在主线的掉落详情）。
  7. 第三场结束解救一位驾驶员 → 引导去抽星舰、教“给星舰装配驾驶员”。
  8. 第四场体验编队，教上阵第二艘星舰；胜利后再解救一名驾驶员。
  9. 第五场遇精英：引导玩家**换不同驾驶员搭配**才能过（不换打不过）——教“驾驶员 × 克制”。
  10. 第 10 关卡住：系统借一颗 NB 星核演示质变（打完收回），勾住核心机制；通关奖励少量星核碎片，并介绍星核与星核碎片。
  > 技能解锁口径补充：为教学即时正反馈，**星舰“首个技能”由早期升级里程碑解锁**（升级既给属性、也在若干等级里程碑解锁/强化基础技能）；**升阶**用于开槽与解锁 / 大幅强化更高级技能。技能仍归星舰，与四层定位不冲突。
- **第 1 天弱引导**：开启 3 天行动、开放打捞港、引导第一次派打捞、展示基地居民变多。
- **第 3 天（首颗永久星核）**：打通**第一个 Boss → 掉落固定“过载核心”（新手核：普攻变“原子炮”——开局一炮、约半个战场范围巨量伤害、10s 一发；非三选一、人人相同）**（成就门控，正常玩家约第 3 天；吃力玩家靠首个行动宝藏奖励助力）。
- **第 7 天**：完成第一次 7 天扩张 → 扩张宝藏**全池自选完整星核** → 正式学习星核装配。

## 12. 跨期追赶（小幅折损）

目标曲线：少玩 1-2 天 → 追回差不多/差一点点；少玩 3-4 天 → 明显比天天玩慢。既不“少玩几天落后很多”，也不“少玩几天无所谓反正追得上”。
机制（多由“上限 + 窗口”自然形成，不加新系统）：
- **离线上限 = 天然追赶曲线**：上限初始 ≈ 1.5 天产出（居住舱可延长），攒满即停、溢出浪费。少玩 1-2 天上线领回接近满上限（仅差一点点）；少玩 3-4 天大量溢出浪费（明显落后）。“上线领一大笔”是爽点不是惩罚。
- 3 天行动 / 7 天扩张 = 窗口补做：少玩 1-2 天剩余时间够补完拿宝库；少玩更久时间不够、可能丢该轮结算。
- 打捞 = 错过即少跑几趟（队伍回来待命不浪费已完成那次，但不自动出新）。
- 不做“假期全额补偿”，确保天天玩的人领先。

## 12.5 存档与持久化（技术口径）

- 核心体验（随用随停、离线累积、忙完回来继续）依赖本地存档。
- **首发：纯本地存档**（微信本地存储），通过**可替换的存档抽象层（SaveProvider）**封装——首发本地实现，未来可平滑换云、不动业务逻辑（呼应在线化不堵死）。
- **正式公开上线前**：评估加一层**微信云存档备份**（仅防丢 / 换设备，不改玩法）。
- **竞技场版**：接服务器权威云（与联网一起做）。
- 存档内容：主线进度、养成、资源货币、背包、活动进度与离线时间戳等。
- 离线收益 = **纯函数（经过秒数, 状态）**：经过秒数在边界计算（首发取设备时钟，未来可换服务器时钟），核心逻辑保持确定、可复算。

## 13. 商业化与广告

- 首发**纯广告**，内购留后续；**预留付费抽象接口**（无副作用、可替换、可离线测试），首发不接真实 IAP/联网 SDK。
- 广告原则：不帮直接过关、不给临时战斗力、不让“不看广告就卡死”；只放在玩家主动想多拿/快一点/更舒服的地方。
- 首发广告点：
  1. **通关奖励看广告翻倍**（不起眼小 ×2 按钮、无次数限制——因首通奖励一次性、重复给得极少，机会天然有限不通胀；作用于通关掉落）。
  2. 赞助补给（看广告 → 补给券，每日上限）。
  3. 打捞加速（看广告减少剩余时间）。
  4. 星辉货舱多选（免费选 1，看广告再选 1）。
  5. 商人小站每日刷新（看广告多刷新 1 次，见 §10.3）。
- **不看广告版本必须成立**：不看广告也能正常推主线、完成 3 天/7 天、正常获得星核、正常形成 5 核编队，只是更慢/选择更少。广告只让体验更舒服。
- 良心红线：不做失败看广告复活/补战力、不做离线收益广告追加、不逼氪、不做“今日必领”式压力文案。

## 14. 首发内容规模（按专业建议重定，不沿用旧“理想/可砍”硬数）

原则：**先用极小原型验证好玩，再按“做得动 + 真好玩”决定首发量**。小而精的垂直切片 > 大而糊的内容堆。

**C1 原型（验证核心循环，色块占位）——最小集**：
- 3 艘星舰（覆盖 2-3 个武器派系）、3 个驾驶员（覆盖 3 个角色）、约 6 条插件效果（跨 3 槽位）、2 个星核、约 8-10 关含 1 个小 Boss、最小基地（船坞 + 居住舱 + 离线产出）。
- 目标：跑通“战斗 → 掉落 → 养成 / 装配 → 变强 → 推进”，验证四层搭配是否好玩。

**首发目标（原型验证后据实调整，给个起步范围）**：
- 驾驶员 6-8（覆盖 5-6 角色）。
- 星舰 10-12（覆盖 6 武器派系；自由 / 流派 / 专属分布）。
- 星核 6-8（够 5 核编队 + 几个可追 / 可收集）。
- 插件：词条库约 18-24 条（跨 3 槽位 × 3 品质，靠合成与品质替换形成成长）。
- 星域 3-4；主线节点约 50-70；精英关 8-10；星域 Boss 3-4。
- 建筑 7：船坞 / 居住舱 / 打捞港 / 商人小站 / 星港补给站 / 研究塔 / 星核展厅。

> 比旧“可砍版”更克制：首发是小团队（你 + 我）+ 4MB 包体，宁可少而打磨透。最终数由 C1 原型的“好玩度 + 产能”反推，不提前拍死。
> ⚠️ 节奏待权衡：内容比旧版缩小（50-70 节点 / 3-4 星域），而 §2 的 21 / 28 天毕业目标未变——第二块节奏表需二选一：要么毕业更早（endgame 更早变薄，靠路线图补），要么战力墙更陡（每节点耗时更长）。两者都可，但要明确选一个。

## 15. 暂不做 / 首发后路线图

**首发暂不做**：每日任务、手动技能/舰长过载、复杂多部位装备、专属芯片、补给密钥、失败看广告过关、强制 PvP、公会、过多战斗词缀、满屏常驻乱炸、星核实验室深层系统、星愿广场、靶场排行榜、复杂商店策略、转生/重置。

**靶场**：移出首发，后期与竞技场一起上线。

**首发后路线图（成功上线后）**：接服务器联网 → 竞技场（优质奖励）→ 更多关卡 → 公会 → 组队挑战副本。长时段深度靠“内容模式广度”逐步支撑（呼应北极星）。
- 在线化预留硬约束：战斗逻辑确定性、配置/种子驱动、可单独跑；核心逻辑不混入真实联网/随机/设备状态；首发不接真实联网。

**未来方向（记一笔）**：参考放置军团出**专属星核**——装在星舰上直接改写其技能效果、发生质变。

## 16. 已锁定核心方向速览

纯自动战斗 · 竖屏上下 · 九宫格上阵 5 舰 · 技能制(取消能量) · 胜利=消灭全部敌人/超时120s判负 · 四层(星舰打什么·驾驶员怎么打·插件调数值·星核质变) · 插件品质制+合成 · 星核三渠道+限定核+第一稀缺 · 基地养成+离线产出(随最高星域提升) · 单一补给券抽卡(只出本体)+脉冲节奏 · 打捞3级信标 · 商人小站(回收+稀有刷新) · 星空宝库定向兑换 · 3天行动/7天扩张 · 无体力 · 本体阶级(星级/字母,无独立稀有度)+抽卡阶级保底 · 站位前中后排/左中右 · 本地存档(预留云) · 纯广告首发(预留内购) · Cocos Creator引擎(TS,战斗核与引擎解耦) · 体验平滑不憋屈。
