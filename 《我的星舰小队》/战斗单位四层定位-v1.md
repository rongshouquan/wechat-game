# 《我的星舰小队》战斗单位四层定位 v1（星舰 / 驾驶员 / 插件 / 星核）

状态：设计稿，待 Ron 确认。目的：把四个组成的职责彻底分开、各有清晰定位、升阶机制互不冲突。

## 一句话分工
- **星舰** = 打什么（武器与技能、基础属性、槽位）。
- **驾驶员** = 怎么打、打谁（目标选择/行为 + 签名能力）。
- **插件** = 某方向补多少数值（纯调校）。
- **星核** = 发生什么质变（规则级变身）。

## 四层定位表
| 层 | 拥有 | 升级/升阶给什么 | 边界（不碰什么） |
|---|---|---|---|
| 星舰 | 基础属性、技能（自带战斗动作：普攻/被动/条件技/CD技）、3 插件槽+1 星核槽 | 升级(星舰合金)→属性；升阶(碎片)→**解锁技能→强化技能** + 开槽 | 不决定目标/行为（那是驾驶员）；不做规则质变（那是星核） |
| 驾驶员 | 驾驶能力（目标选择/行为 AI）+ 驾驶天赋（独特被动） | 升级→强化**驾驶天赋**数值；升星→解锁/深化**驾驶能力**+天赋质变（**无原始属性**） | 不自带独立技能/大招（大招归星舰）；只“改写/增幅”星舰已有的技能、普攻、目标 |
| 插件 | 定向数值/小效果（品质制：精良/优秀/传奇） | 不分等级；换更高品质 / 合成升品 | 只“微调数值”，不解锁技能、不改行为、不质变 |
| 星核 | 规则级质变效果 | 获取即质变（非常规养成；一船一核） | 最稀有、最夸张、最视觉的变身 |

## 关键区分：星舰“技能” vs 驾驶员“能力”
- **星舰技能** = 这艘船自带的战斗动作（独立触发：被动/条件/CD）。升阶解锁更多技能、强化技能数值与效果。
- **驾驶员能力** = 改写/增幅“打法、行为、星舰技能的表现与目标”，**不自己独立放一个技能**。升阶解锁能力、再升阶强化能力。
- 一句话：星舰技能是“动作本体”，驾驶员能力是“怎么用这些动作、打谁、打出什么味道”。

## 一个例子（导弹舰，把四层串起来）
- **星舰**（导弹舰）：自带技能“齐射”（CD技：发一排导弹）+ 被动“装填”（普攻概率多打一发）。升阶→解锁第二技能/强化齐射。
- **驾驶员·突袭手** 能力“猎杀后排”：改写齐射目标→优先锁后排；升阶→对后排额外 +X% 伤害。
- **驾驶员·炮手** 能力“过热”：持续输出叠加伤害；升阶→提高叠层上限。
- **插件**（武器·传奇）：+暴击率，暴击时溅射（纯数值/小效果）。
- **星核**（小太阳）：把“齐射”质变成“发射一颗吸附敌人后爆炸的小太阳”（规则级质变）。
→ 同一艘船：星舰给动作、驾驶员决定打谁/怎么打、插件补数值、星核变身，四层不重叠。

## 升阶机制对照（都用“解锁→强化”，但作用对象不同）
- 星舰升阶：解锁/强化**技能** + 阶段性开槽（插件槽 1/2/3、星核槽）。材料=星舰碎片（专属，抽卡重复）+ 通用星舰碎片（活动）。
- 驾驶员升级（驾驶记录）：强化**驾驶天赋**数值（无原始属性）。
- 驾驶员升星（驾驶员碎片 + 通用驾驶员碎片）：解锁/深化**驾驶能力** + 驾驶天赋质变。
- 插件：不升阶，换品质 / 合成升品。
- 星核：不升阶，获取即用（首发不做重复星核深层养成）。
