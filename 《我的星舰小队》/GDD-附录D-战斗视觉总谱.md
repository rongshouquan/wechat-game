## §0 第一性原则（四条）

1. **战斗=一场约 25 秒的烟花秀**（时长靶 GDD §4：常规≈25s / 碾压≈10s / 精英≈35s / Boss 45-60s）——演出必须在这个窗口内完成"入场→交火→高潮→收束"，没有慢热的资格。
2. **看得懂优先于看得爽**："观察战斗、找搭配策略"是核心体验（秒办哲学修订）——读法三定：**形状=干什么**（弹种）、**颜色=谁打的**（我方按舰主色 / 敌方按族群色）、**大小=多重要**（V 档）。（2026-07-13 签名制补充：我方**伤害弹**的形状升格为"谁打的"第二通道——每舰一个签名形状；治疗/护盾/增减益等**功能弹**形状仍全局统一=干什么。详见 §2a。）
3. ~~**响度预算制**：爽感靠对比不靠堆量——平时克制、关键时刻震撼~~ → **高潮节奏制（2026-07-16 Ron 拍·两竞品真机录屏对标实证〔放置军团/百炼英雄=殊途同归的"高潮常态化"：大排场 3-5 秒一轮=战斗的呼吸节奏〕）**：爽感靠**持续的节奏化高潮**——技能（V2）演出全面加料（每个技能配得上"小高潮"排场）、全屏级特效以 3-5 秒一轮的节奏出现；**星核质变（V3）仍保有更高一档的排场**（炸有层次，稀缺不再）；震屏白名单维持不变（震靠排场不靠晃）。战斗中 UI 维持轻装（学放置军团纯净战场，不学百炼满配——我们是关卡制非挂机大世界）。
4. **资产纯净体**：单位图不带任何炮口光/弹道/命中光效/尾焰（附录A A5/A6 通则·24 批拍板收口）——战斗的全部"响"由引擎用四件套+特效层现场合成。

## §2 弹种库与"一发炮弹的解剖"（四件套）

**四件套**：每次开火 = ①**炮口闪**（0.1s·弹色）→ ②**弹体+拖尾**（弹种形状 × 发射者颜色；拖尾=弹体拉伸+透明渐变，不单出图）→ ③**命中爆点**（0.15-0.3s·按 V 档取尺寸）→ ④**余韵**（可选：灼痕光斑/雾团残留/电花·≤0.5s）。
**产能账**：美术只产一张通用特效图集——炮口闪 ×3 型、爆点 ×3 档、糖果粒/护盾泡/状态小图标 + 弹体件（~~14 弹种弹体~~ 2026-07-13 签名制后=我方 20 签名弹体〔部分共用零件底图换色/换节奏〕+ 敌方族弹体），合计 **≈45-60 张小图**（原 ≈30-40）；星核 16 颗多数复用"弹种放大+换色"，仅 **5 颗形状级质变出专属件**（陨星弹/小太阳/星鲸/曲率星门/超新星——清单定死，见 §2b）。

### §2a 我方签名攻击制（20 舰"特效即角色" · 2026-07-13 Ron 拍板）

> 对标《荒野乱斗》：**特效跟角色绑定**——看一眼弹道就知道是哪艘舰在开火。制度基础＝星舰真源每舰"开火签名"字段（雏形已在），本节把它升格为制度并补齐规格；**签名逐舰内容仍以星舰真源为真源**，本节只管制度、预算与验收。

- **两层制（与敌方"阵营语法两层制"对称）**：**族基调层**（五职业族各一个弹道性格，同族共享"味"）＋**个体变奏层**（同族 4 舰在基调内各自成签名）。族基调自星舰真源 20 舰开火签名归纳（非新发明）：

| 职业族 | 弹道基调（性格） | 真源体现 |
|---|---|---|
| 突击 | 快、小、密——连发/点射/蓄力狙的"哒哒哒"系 | 极焰高频小弹 / 锋矢三连点射 / 影刃蓄力长狙 / 蜂针点射+延迟雷 |
| 护卫 | 短程、厚重、环与盾形——"哐"系 | 磐石旋飞盾片 / 铁壁震荡环前推 / 堡垒慢重弹 / 山岳震荡波 |
| 炮击 | 大颗慢弹/常亮射线/弹幕规模——"轰"系 | 烈阳全队最大慢弹 / 群蜂导弹雨 / 贯日唯一常亮射线 / 霹雳弹跳电球 |
| 支援 | 柔和光弹飞向友军、飘与洒——"抚"系 | 晨曦盾泡 / 号角旗光片 / 甘霖水珠 / 春风光点洒片 |
| 工程 | 放置物与异形弹（雾/笼/机/盒）——"摆"系 | 迷雾雾团 / 蜂巢无人机 / 哨卫钉弹+诱饵盒 / 锁链缠电球+牢笼 |

- **签名四要素（建档规格·星舰真源"开火签名"字段按此补齐）**：①**形状**（弹体外形，取自/变奏自本节零件库）②**数量与节奏**（单发/连发/簇/蓄力——开火的"鼓点"）③**发光色**（=舰主色，沿用颜色语法）④**落点行为**（命中后那 0.3s 的戏：溅射环/延迟"啵"爆/弹跳拉弧/贯穿留痕/罩泡/落格放置……）。现字段已含①②③，**④为本次新增维度**，随签名精雕批逐舰补写入星舰真源。
- **签名验收测**：遮住舰体只看弹幕，**3 秒认出是哪艘舰在开火**（对齐单位类"35% 三秒测"的弹药版）。
- **功能色不让位（铁）**：治疗恒绿/护盾恒蓝/增益金黄/减益紫沿用——**签名个性只落在伤害弹与放置物上**；支援舰个性走载体形状与节奏（泡/旗/珠/瓣各异），颜色仍守功能色。
- **响度守恒（铁）**：签名=**形状与节奏的个性，不是尺寸与亮度的个性**——全部签名在 V1/V2 预算内表达，不得借签名膨胀成 V3 排场（质变排场仍星核独占，防响度通胀）。
- **分批精雕**：首批签名试作（每族 1 艘·舰单候 Ron 拍）四要素做满→验收方法与观感→其余 15 舰分批补齐；未精雕舰过渡期＝族基调+舰色+真源签名简化版。
- **敌方/召唤物不适用**：敌方守族语法两层制（族弹种×族色），召唤物用弹种库原件——个体签名是我方 20 舰的专属待遇（玩家的收集对象才配"角色级"特效）。

**统一口径（原样并入）**：
- "下一次攻击 / 下一发"一律写明是**普攻**还是**技能**（如"下一次**普攻**分裂成 2 发"）。
- 范围词用通用定义：单体 / 小范围(十字5格) / 一片(3×3) / 一列 / 一直线 / 全体 / 友方同理。
- 被动若不发射弹种（如光环/护盾/形态），直接写效果即可，不用弹种。

**颜色语法**：
- **我方弹 = 本舰主色/开火签名发光色**（`星舰真源`各舰视觉档案；造型三版舰体走糖果家族壳+每舰本色、弹药色跟舰主色走当个体识别的第二重通道——2026-07-07 口径同步），**敌方弹 = 族群色**（星盗糖果红 / 无人舰冷蓝紫 / 废铁锈橙 / 污染紫绿）——一眼扫过知道"谁的火力在起作用"。
- **全阵营通用功能色**：治疗恒绿、护盾恒蓝、增益金黄、减益紫（功能色 > 身份色，不随发射者变）。

## §3 标准反馈五件套（伤害数字瀑布制 · 2026-07-16 Ron 拍板转正）

## §4 背景动效层（Ron 提案 · 2026-07-07 拍板⑧立项）

> 治"固定阵位=棋盘感"的宏观动：**星流让整场战斗读成"我方全速追击敌舰队、边追边打"**（竖屏、我方朝上敌方朝下的既定布局天然成立）。纯视图层，零改战斗逻辑。

- **三层结构**：
  - **L1 远景星云底** = 拍板⑤的 **4 套敌人族群基调背景 + 色调变体覆盖 6 星域**（星盗浆果糖霞 / 无人舰冷蓝机械带 / 废铁锈橙尘带 / 污染紫绿雾域·均为亮调）；静态或极慢漂移；**明亮律（Ron 2026-07-07 拍板·32/34 批实证）**：整体明亮清澈（对齐"鲜艳清澈透亮"），星云/尘带只作小面积点缀**不得铺满背景**，禁深空黑/夜空基调；**但明亮≠过曝——柔和均匀光、避免死白高光/整片曝光感，亮而通透不刺眼**。
  - **L2 中层程序星点流**：代码生成小亮点（池 80-120 个·同贴图合批≈1 个 draw call）恒向下流动——**0KB 资产**。
  - **L3 近层稀疏道具**：彗星/碎石/远行星侧掠，数秒一个、池化复用。
- **速度情绪曲线**（一个速度变量当情绪乐器）：

| 时刻 | 星流 |
|---|---|
| 入场仪式 | 快掠——舰队全速赶到战场，与"从屏下滑入 0.5s"咬合 |
| 常规交火 | 慢速巡航——稳定"在飞"感，不抢戏 |
| V3 质变触发 | 星光瞬间拉成线（曲速感）0.3s 脉冲，与压暗+推镜叠加 |

| 胜利 | 加速，我方依次冲出画面上缘——追击继续 |

- **三护栏**：①动效元素一律**低饱和低对比封顶**（"背景让位、单位吃全画面最高饱和"铁律对动效同样生效）；②默认速度=巡航，快只在脉冲时刻；③我方单位脚下一圈**极淡光晕锚定**，滚动背景下单位永远"钉得住"。

### §4a 战场板构图与高度语法（组装 v6-v8 定案 · 2026-07-14 Ron 拍板"敌我平行空中对打"）
