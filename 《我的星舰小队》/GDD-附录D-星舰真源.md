## §0 总则

**视觉档案（2026-07-07 Ron 拍板②·取代旧一行式"外形"；同日两次实证迭代至**造型三版**）**：每舰五字段＝**剪影** / **配色** / **锚点**（识别记号 1-2 个，3/4 与俯视两视角都必须保留）/ **开火签名**（弹种×节奏×发光色，引总谱 §2；**弹色=开火签名所记发光色**）/ **禁**。**造型三版三律（33 批海报实证推翻二版"制服"路线·全局语法与"35% 三秒测试"=附录A A3）**：①**家族脸（统一靠它）**——20 舰共享"标准星舰体"底模＝胖圆糖果壳 / 一颗同款大圆泡舱〔有色玻璃·无人·"眼睛"级识别符〕/ 对称短翼或引擎舱 / 尾部青蓝喷口 / 糖果光泽渲染；②**每舰本色（撤制服）**——各舰主色回归自成一格〔五颜六色 OK·同型 4 舰 hue 互错·统一不靠同色靠家族脸〕，下方各舰"配色=糖果家族壳＋主色 X"即此；③**约束少而精**——类型差异走"比例 + 装备件"、装备件不压过舰体（防球化/枪化/环包身）；旧"夸张剪影"（盾占整脸/炮占全长/光环包身/球壳/塔形舰）与"舰队制服色"均作废、尖锐词作废照旧。

**层级铁律**："造成/施加"归星舰，"利用/放大"归驾驶员。星舰技能 = 具体动作（开火/范围/群体/召唤/护盾/治疗/增益/施加状态），**不指定打谁**（打谁由驾驶员定，默认最前排）；星舰被动只塑造"本舰自己怎么开火/什么形态"，不写"对某类敌人增伤"。

**作用形态术语**：单体=1格 / 小范围=目标+十字4格(共5格) / 一片=目标+周围8格(3×3) / 一列=同列5格 / 一直线=本舰与目标连线穿过的格 / 全体=所有敌人 / 友方同理 / 全队=我方全部。**残血** = 血量 <30%。

**召唤物规则**：召唤物在**离召唤源最近的随机空格**生成、**各占 1 格**、不移动、原地攻击、默认打最前排；**召唤源死亡/超时/被打掉则其召唤物一并消失**；召唤物即普通可攻击单位（按所在格被单体/范围攻击）。敌我同理。

**状态/控制库**：硬控=短路(不能动)/沉默(不能放技能)；软控减益=减速/虚弱/破防/致盲/易伤/燃烧(持续掉血)/削盾/驱散；增益=加攻/加攻速/护盾/持续回血/增伤/净化。**平衡**：硬控递减 + Boss/精英控制抗性。

**触发口径表（术语定义 · 2026-07-03 补写死）**：

| 术语 | 口径 |
|---|---|
| "重击" | 泛指**技能命中**（区别于普攻）；技能命中即算重击，普攻不算（如铁壁号「坚甲」"受重击时叠一层减伤"＝受到任意技能伤害时触发，普攻不触发） |
| "停火蓄力" | 技能描述含"蓄力/蓄能"时，本舰**当帧暂停一次普攻**改放技能（技能与普攻同一时刻二选一，不叠加触发）；蓄力**不可被打断**（自动战斗无玩家操作介入，仅敌方沉默类效果能阻止其释放） |
| "攒满"类技能速率 | 群蜂号「饱和打击」这类"蓄力：每消灭1敌+1层，满N层放"的技能，层数**只由指定事件驱动**（如击杀数），不随时间被动增长；技能释放后层数清零重新累计 |
| "命中"含友方 | 治疗/护盾/增益类范围效果的"全队/友方一片"目标**默认包含施法本舰自己**（如晨曦号「圣盾」的"全队"含晨曦号自己）；伤害类技能的目标只面向敌方，不会误伤友方 |
| 曲率星门 vs 多格 Boss | 曲率星门"开局一次性整排对调"（见星核真源）**只对占 1 格的单位生效**；Boss 常占多格（`sizeRows`/`sizeCols`>1），**不参与整排对调** |
| 开槽顺序 | 单舰 3 插件槽固定解锁顺序＝**武器槽（C阶起）→ 技能槽（B阶起）→ 战术槽（A阶起）**，装配界面按此顺序从左到右排列（对齐附录B B1.4"开槽 gating"） |

**基础暴击口径（2026-07-03 补）**：全体单位基础暴击率 **5%**、暴击伤害倍率 **150%**（暴击造成 1.5 倍伤害），数值校准期可整体上调/下调，此处只定基准（0% 起点会让"暴击率+"类天赋失去意义，5% 给个有感的基础频率）。

**成长模板（星舰 · 三条线都有奔头）**：
- **起手 Lv0（只普攻）**；Lv1 解锁技能1、Lv10 解锁技能2；每级长属性 HP/攻击；**每 20 级（Lv20/40/60/80/100）一个"升级大节点"强化**（数值/小机制，非质变）。
- **升阶（专属碎片，数字暂定：30 = 攒够合成本体〔背包碎片合成出该单位 C 级本体，见 S10.1；已直接拥有本体的不花这笔〕，50/100/300/1000 = 四次升阶消耗〔C→B=50 / B→A=100 / A→S=300 / S→SS=1000，按顺序各花一次，2026-07-03 读法写死〕）**：C(1插件槽) → B(2槽) → A(3槽) → S(开星核槽) → SS；开等级上限 20/40/60/80/100、每阶全属性跳；**A阶=大质变 / SS阶=终极质变**（B/S 阶主给槽+属性，每舰下文写明）。

## §1 突击舰（刺客 · 低血/高攻速/高单体爆发）

### 极焰号
- 视觉档案：剪影=小巧圆墩炮艇——窄机身+泡形座舱+短粗双翼，鼻尖一门粗短主炮（≤半身），尾部双大圆喷口；配色=糖果家族壳＋主色草莓红（机首环带+部件）；锚点=鼻尖短炮+双喷口；开火签名=高频草莓红小弹（直射炮弹），集火炮=三倍大的一发；禁=细长尖角、炮超半身。属性(C·Lv0)：HP 800/攻 60/攻速 1.0/防 20。
- 普攻：发射【直射炮弹】打单体 ×1.0。
- 技能1「集火炮」(主动·CD8s)：对单体 ×3.0。
- 技能2「快速装填」(被动)：普攻 20% 概率立刻再打一发。
- 升级：Lv0只普攻→Lv1集火炮→Lv10快速装填→每级长属性；大节点 Lv20 集火炮溅射相邻4格(1/3) · Lv40 装填→40%连发 · Lv60 集火炮溅射扩相邻8格(1/3) · Lv80 装填→60%连发 · Lv100 集火炮溅射→相邻8格(2/3)。
- 升阶：C(1槽) · B(2槽+属性跳) · **A(3槽·大质变：集火炮伤害+100%)** · S(星核槽·装填→80%连发) · **SS(终极质变：集火炮连续释放三次)**。

### 影刃号 ★（绑「影」）
- 视觉档案：剪影=全队最窄小的轻快艇——前倾姿态+一对不成比例的大引擎，机腹一门修长狙击炮（不超机身长），背部一对短圆鳍；配色=糖果家族壳＋主色深莓紫蓝（deep berry/grape-blue·比蓝莓更深更偏紫·侧带+座舱圈·甜暗糖果调，非整舰深色）；锚点=腹下修长炮+大引擎；开火签名=长蓄力→全场最细长的青蓝粗束，残影=普攻分裂双束；禁=整舰深色、金属黑、流线尖锐、炮超机身。属性(C·Lv0)：HP 650/攻 75/攻速 1.3/防 12。
- 普攻：发射【聚能束弹】打单体 ×1.0。
- 技能1「狙杀」(主动·CD7s)：蓄能射出一道极粗聚能束，对单体 ×4.0。
- 技能2「残影」(被动)：每 3 次普攻后，下一次普攻分裂成 2 发聚能束打 2 个目标。
- 升级：Lv0普攻→Lv1狙杀→Lv10残影→每级长属性；大节点 Lv20 狙杀×4.5 · Lv40 狙杀×5.0 · Lv60 残影改每2次普攻触发 · Lv80 狙杀×6.0 · Lv100 分裂每发打3目标。
- 升阶：C(1槽) · B(2槽+属性跳) · **A(3槽·大质变：狙杀命中后若目标存活则立即再射一道)** · S(星核槽·残影分裂→3发3目标) · **SS(终极质变：狙杀改为连续狙击3个不同目标)**。
- 专属（影驾驶）：狙杀分裂成 2 道、同时狙 2 个目标。

### 锋矢号

### 蜂针号
- 视觉档案：剪影=圆滚小艇——短翼+尾部一根圆头锥刺发射器（≤半身）；配色=糖果家族壳＋主色蜜橙（2-3 道环纹·深可可细环）；锚点=尾锥刺+环纹；开火签名=蜜橙细束点射，定时爆破=贴身闪烁小导弹延迟"啵"爆；禁=真昆虫肢体、黑黄警示条、刺超半身。属性(C·Lv0)：HP 680/攻 72/攻速 1.25/防 13。

## §2 护卫舰（坦克 · 高血高防）
