## §1 六段式骨架（正文=固定顺序，每段 1-2 句，只写"是什么"）

| 段 | 写什么 | 铁规 |
|---|---|---|
| ① 触发+主体 | `xjxd style,` + 单位/画面名 + **强锚 ≥3 个**（自带形象的硬名词：皇冠/王旗/巨炮/歪帽/瞭望塔…） | **禁相对词**（"中型/收敛/介于之间/比X豪华"=空气，模型必漂）；锚不足→先跑造型海选批找形象 |
| ② 等级/职能语言 | 用**可数的具体量**表达级别：旗数、炮数、豪华件规格（大王三旗满炮 > 大副单旗单炮） | 数量词写死（ONE/THREE），不写"多/少/一些" |
| ③ 构图视角 | 资产=STRICT top-down orthographic + 机头朝下 + 画幅占比；场景/海报=各自框架句 | 视角词+斜视角进负面口双保险；**低保留度垫图改不了视角**（要换视角=改图刀转机位） |
| ④ 配色锁 | 族色主导色点名 + "gold used ONLY as small trims" 式限定 | 反向色写进负面口（如 golden hull）；不写抽象色令（"主色调改X"=滤镜效应老坑） |
| ⑤ 画种渲染 | 2D 手绘声明（hand-painted 2D game illustration, flat cel shading, NOT a 3D render）+ 锁样锚词族 | **chibi 只修饰人**，永不修饰载具/建筑（万物给脸真凶）；"glossy/polished"单独出现会引 3D，必须伴随 2D 声明 |
| ⑥ 背景/资产收尾 | single game unit + 净灰底句 + no scene + 图中文字用**引号包裹**（"星舰小队"） | 加 "game asset/sprite" 类资产信号词防画成概念插画 |

**人物条款**（有人物必写）：EXACTLY ONE …figure（多人进负面口）；动作=具体在干什么（扶栏/持望远镜/掌舵），**禁敬礼/挥手**；脸朝镜头可（萌点），但不许"船朝下人朝侧"的拧巴向。

## §2 负面口标准库（禁令只进这里，正文永远不写 "no XX"）

- **通用库**（每批必带）：3D render, CGI, blender render, octane render, realistic lighting, ambient occlusion, depth of field, photorealistic, watermark, text, logo, UI, panel lines, rivets, dark mood
- **单位题追加**：multiple crew members, many figures, crowd, cartoon face on vehicle hull, eyes and mouth on spaceship, creature inside cockpit, ocean/sea/water/waves/splash, torch/fire/flames, engine flame plume, projectile, muzzle flash/glow, halo, cast shadow/drop shadow, floor/ground plane, background scene/texture, isometric view, 3/4 view, side view, horizon
- **场景题追加**：giant smiling star mascot, face on building, smartphone/phone frame/device bezel, harbor/dock/pier
- **按病加词**：新病先查"引子词黑名单"（§4），**先删正文引子再谈负面**。

## §3 参数基线与生成模式

- **文生图（原生）**：20步 / cfg 3.2 / euler+simple / shift 3.1 / 滤镜 xjxd_style_v1 @1.0；官方金配置（50步/cfg4.5）留作"终选精修档"，用前 A/B 验证不伤滤镜味。
- **垫图**：d0.35-0.45=族内翻新（继承构图配色）；d0.55-0.60=借形改造；**垫图会继承参考的一切（含病与视角）**——参考有病先治参考（改图刀除根一次）。
- **改图刀（Qwen-Edit）**：探索/应急三用途=净病灶/单点修/转机位；**正式资产禁多重手术链**（原生律 Q8）。
- **正式资产=单次原生生成**；抠底后交付；展示拼版由程序统一垫同色浅灰底（生成端只加底色锁词不硬追）。

## §4 引子词黑名单（实证追加制·写词前扫一遍）

| 引子词 | 会引来 | 替代写法 |
|---|---|---|
| mobile game poster / 手机游戏海报 | 手机壳边框 | vertical key art illustration, full-bleed |
| port / 港 | 海面海港 | floating sky base / drifts in deep space |
| pirate boat（孤用） | 水面+火把 | space-pirate hovercraft, hovering in empty void |
| chibi toy style | 万物长脸 | candy-cute cozy sci-fi（chibi 只给人） |
| 抽象色令（主色调改X） | 滤镜效应 | 具体部位+具体色 / 情绪词控色 |
| sailing ship 联想词（sail 慎用） | 帆船海洋感 | banner on mast（旗非帆）·用前过"禁帆船"红线 |
| 相对词（grander/humbler/中型） | 形象漂移 | 可数硬件（N旗/N炮/件名） |
| holding spyglass up to his eye（举镜到眼） | 人物必侧头侧身=撞朝向规范 | 正对姿势动作（双手扶栏/掌舵/按操作台，面朝镜头或机头） |
| guardian-angel 词族 | 长羽毛天使翅（L20 实证） | 晨曦=Ron 拍"好方向"有意用；其他单位慎用（要光环写 golden ring 物件） |
| 全大写舰名（PANSHI 类） | 名字被印上船壳（L21 实证） | 要印字走"引号包裹"规则；不要印字=删名或写小写普通词 |
| 色名带水果（plum-grape 类） | 长出真水果挂船上（L24 实证：船尾长真葡萄串） | 加 tone/toned 后缀（deep plum-violet tone）或用非果名色词 |
| 非常规剪影靠纯文字（前指翼/舰体前齿） | 被"飞机=后掠翼"先验吸收（L24/L25 三战术全败） | 剪影级改造走垫图借形 d0.55-0.65 或改图刀"保剪影换皮" |

## §5 发批前强制检查单（逐项打勾贴进脚本头）

1. ☐ 强锚 ≥3 个硬名词，零相对词
2. ☐ 等级/数量全用可数词写死（ONE/THREE…）
3. ☐ 正文零 "no XX"，禁令全在负面口
4. ☐ 负面口=通用库+题型追加库齐全
5. ☐ 视角框架句+斜视角负面双保险
6. ☐ 族色锁+反向色负面
7. ☐ 2D 声明在场；chibi 只修饰人
8. ☐ 人物条款（独员令/动作具体/禁敬礼）
9. ☐ 引子词黑名单扫描通过；图中文字已用引号包裹
10. ☐ 滤镜挂载+参数=基线（或注明偏离原因）；多抽 ≥3

## §6 风格词库（糖果Q萌专项 · 2026-07-10 网查印证自家词库）

- **实证核心词（生产用·全部经 L 批验证）**：candy-cute cozy sci-fi / soft cel shading / glossy candy highlights（必须伴随 2D 声明）/ rounded one-piece smooth shell / thick dark same-hue outlines / big clean color blocks / 锁样色盘（mint cyan·sky blue·lemon yellow·candy orange·cream white·dreamy light purple·soft pink）/ chibi（**只修饰人**）+ 2-3 heads tall。
- **候选词（网查所得·未实测·上生产前先小批验证）**：人物专用=blush marks、sparkly oversized eyes、super-deformed proportions；通用=gentle shading、clean linework。
- **风格新雷（网查+本项目雷区合并·禁用）**：sticker / die-cut sticker（引白边——L12 调色刀白边同族）；3D vinyl collectible / blind box figure（引 3D 感）；big eyes 等五官词用于载具/建筑（长脸真凶族）。
