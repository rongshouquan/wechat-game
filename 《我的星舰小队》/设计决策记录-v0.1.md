## 1. 离线 / 挂机产出模型 ✅ 已确认（方案 2 弱化版）

补掉原方案最大的循环断点。骨架如下：

- **基地被动产出**基础资源（星矿 / 合金 / 战斗记录），离线累积，**上限由居住舱决定**；上线一次性“领取离线积累”，保留“哇攒了一大笔”的领取仪式感。
- **产出速率 = 建筑等级基础 × 星域系数**；星域系数由“已通关最高星域 / 节点”决定——**推主线越远，离线产出基线越高**。
- 推主线因此有双重正反馈：① 首通一次性奖励；② 永久抬高离线基线。
- **重复挑战仍不刷资源**（沿用原口径，不破坏）；不做“挂机刷当前关”的战斗画面。
- 打捞 = 额外的定时离线产出（信标 → 稀有资源）。
- 卡关时玩家变强的“分钟级”手段 = 领离线积累 → 养成 → 再战；配合无体力（见 §9）可免费重试。

## 星核设计支柱（玩法核心，单列）

- 星核 = 本作“宝物”，对标放置军团的宝物，是**核心玩法与中期爽点引擎**，不是普通养成项。首发几个，长期做到十几二十个。
- 玩家短期爽点闭环：**卡关 → 渴望 → 获得牛逼星核 → 兵种质变、视觉更爽 → 碾压推好几关 → 再卡 → 再渴望**。
- 据此校准、指导后续数值的两条原则：
  - **稀缺性落在“拿到指定的 / 最强的那个”，而非“获得星核本身”**。获得本身要有一定流速做爽点引擎，否则中期“渴望”转不动，会变成干等。
  - **卡关墙的长短是数值活**：长到能制造渴望，短到不至流失——这块可调。
- 节奏表待办：随机星核出处要够（不能只有每 7 天一次扩张宝库），否则“随机为主”名不副实；可考虑部分精英/Boss 首通、特定信标小概率直接出星核（而非只出星空宝石）。

## 战斗：技能制（取消能量）✅ 已确认

- **取消能量条。** 术语从“大招”改为“技能”（“大招”暗示个个威力巨大、易致开场齐爆，弃用）。
- **胜负判定**：唯一胜利条件 = 消灭全部敌人；我方全灭 / 时间耗尽 / 其它情况 → 一律判负。
- **技能采用异构触发（三类）**（学放置军团，靠触发多样性天然避免“5 艘齐爆”）：
  - 被动型：常驻，如强化普攻、光环。
  - 条件型：满足条件触发，如“击杀 N 个单位后”“某方阵亡到某数”“己方血量低于 X%”。
  - CD 型：每隔 CD 自动释放，区间约 10–30s（弱招短、强/长效招长）；**默认开局即放第一次**，故不单设“开局型”。
  - → 避免齐爆的关键是**把星舰技能与驾驶员效果设计好**，让节奏自然错开，而非靠统一 CD 硬错峰。
- 关卡时长目标（1x，配合 2x/3x 倍速压缩）：
  - 战力匹配、正常推进：单关 ~20–30s。
  - 战力碾压（越级/回刷）：开场一波 ~5–10s 秒清。
  - 卡关：打到 120s 超时判负（= 失败提示“超时未击破”）。
  - Boss：~45–90s，含 2–3 阶段。
- 连带修订：原“能量条”UI → 改为技能就绪 / CD 指示；驾驶员“工程师=加速能量”→ 改为“缩短技能 CD / 加速首发”；词缀“短路/打断”= 一段时间无法触发技能。
- 未来方向（记一笔）：可参考放置军团出**专属星核**——装在星舰上直接改写其技能效果、发生质变。

## ✅ 第一颗星核与限定核（已确认）

- **第一颗永久星核 = 第一个 Boss 首杀掉落“星核宝藏”，开箱三选一**（成就门控，正常玩家约第 3 天打通第一个 Boss；会玩/肝一点 D2–D3，普通玩家稍慢）。比固定“第 3 天”更好——奖励绑成就而非日历，更有“挣来的”感。
  - D1 试用：编入新手教程，遇到难关时“系统借一颗 NB 星核”给玩家体验质变，打完收回。
  - D7：扩张宝库首次 = 全池自选（更大的“随便挑”里程碑）。
- **星空宝库分层池 + 限定核**：通用核随机/兑换都能拿；**2–3 个最强/最酷的为宝库限定（只能宝石兑换、随机开不出）**。护栏：限定核可强可酷，但不做“通关必需”、不做付费墙。
- 后期循环（记一笔）：收集长尾 + 追限定核 → 按关卡/未来竞技场换核 build；池子扩大后随机自然稀释、定向兑换价值自升；后期内容靠路线图承接。

## ✅ 驾驶员定位（已确认）

问题：本作为“驾驶员+星舰+星核”，比放置军团“军团+宝物”多一层。多这层必须值，否则不如砍。

评估：之前“驾驶员决定怎么打”方向对，但描述偏模糊，易与星核（改技能）/插件（调数值）功能重叠——这正是“做不好不如摒弃”的风险点。

建议把驾驶员钉死在一个别人不碰的层：

- **战术大脑 = 目标选择 / 行为 AI**（打前排/后排/最肥/最弱、保护谁）。星核和插件都不管这一层，是驾驶员不可替代的价值。
- **一条签名天赋**（改变“怎么打”的独特被动，如炮手“越打越热”、突袭手“专斩后排/高血量”、工程师“技能命中奶全队”、干扰手“普攻附带短路”）。可挂到新技能模型上（驾驶员改/加技能的触发与目标）。
- 仅少量数值倾向，不做主力（否则与插件重叠）。
- 硬标准（防退化成数值棒）：**每个驾驶员必须一句话讲清且诱人**，讲不清的不做。
- 保持轻量：不强制每艘船都纠结配谁；适配是给爱钻研者的深度，不是所有人的必做功课（这是“多一层却仍轻量”的关键）。
- 与星核划界：**驾驶员管“怎么打/打谁”+被动；星核管“技能本身质变”**（更夸张、更视觉、更稀有）。星核仍是最闪的质变层。

Ron 拍板：按此“战术大脑 + 签名天赋”定义保留驾驶员。

## 已识别的精简点与风险（本轮评审，待处理 / 待验证）

### 可精简 / 冗余
- **🔧 能源槽失义（必须修）**：取消能量后，插件“能源槽（能量/释放节奏）”失去意义。重定义为“技能/CD 槽”（缩 CD、加速首发、改触发节奏）或并槽。3 槽结构不变，只改 2 号槽身份。
- **💱 货币/资源种类偏多（与“轻量”冲突）**：星矿/合金/战斗记录/太空补给/星贝/星核碎片/星空宝石/驾驶员碎片/星舰图纸/改装图纸/招募券/整备券/补给券 + 5 类信标 + 4 类箱子。建议一次合并评估：① 招募券/整备券/补给券是否并成“一种券 + 选池”；② 取消奇迹信标（仅服务 2 个奇迹建筑）；③ 碎片 vs 宝石双币保持但错峰登场。
- **🏛 奇迹建筑偏重**：为 2 个建筑配整套“奇迹信标/蓝图”经济过重。建议星核展厅（贴合星核收藏核心）保留，观星台效果并入研究塔/打捞港，取消奇迹信标。
- **🎯 靶场**：原型不做；首发若做须确认它不是“只为肝帝服务的负担”。

### 风险隐患

### 星港补给站取舍（倾向取消，待 Ron 最终点头）
- 关键判断：本作最大爽点（星核=宝物）明确不在抽卡池；抽卡池只产驾驶员/星舰/插件这类“非核心层”，且这些已有大量其他获取渠道。故抽卡系统在本作只增系统重量、不掌核心爽点，接近冗余。

### 主动放下 vs 被动放下（专业分析 + 建议，方向待 Ron 拍）
- 放置/养成类天生形态 = “被动放下 / 短时多次 / 每日回归”：靠每日回归与长期成长留人，不靠长时段马拉松；其内容深度来自“多内容模式并行”，不是主线无限刷。
- “消消乐式无限刷主线”与本作核心冲突：会摧毁“卡关→渴望→获得星核→质变→爽推”循环（无墙则星核无意义），并导致资源/战力无限通胀。故不建议主线无限可刷。
- 支撑“喜欢的玩家每天玩 3-4 小时”的正确做法 = 内容模式广度（竞技场/副本/爬塔/活动…），各有每日额度、合起来够玩——正是 Ron 路线图方向，属长线。
- 首发（原型先行、4MB、纯广告）：顺应“短时多次”天然形态，不为未验证首发硬造无限内容。把“墙”做平滑 + 离线/打捞/周期产出足够频繁，让“干完今天的→自然收手”像主动收手而非撞墙。
- 建议结论：首发偏“被动放下但体验平滑”；长线靠模式广度逐步支撑“主动放下 / 长时段”。
