## §0 使命与边界

- **使命（Ron 2026-07-08 开场令）**：美术生产从"云端慢、画风飘"换成"本地快、画风稳"。四块交付：①本地出图管线 ②《全游戏视觉表》（龙头）③"什么效果用什么最省做法"对照表 ④骨骼动画评估。外加两贯穿件：本总纲、素材规格与体积预算（=附录A A6，2026-07-09 升级版）。
- **总路线四步（Ron 2026-07-09 拍板）**：管线成型 → 省做法对照表+骨骼评估（一份文档）→ 全游戏视觉表（分批拍板）→ 按表量产。
- **边界铁则**：美术总控只动 美术文档／管线／附录A／视觉表，**不碰 `game/` 工程代码与数值文档**；"好不好看/是不是这个味"永远 Ron 拍板，总控只预筛、出方案、给专业建议。双总控并行规则见 README 活线头节。

## §3 画风真源与方法论（指针）

- 唯一锁样＝**第15批版B-抽3**＋复现配方（`美术素材/锁样定稿/`）；渲染语言/配色/形象语法＝附录A v2；单位身份词＝附录D 各真源；战斗演出＝总谱。
- 提示词方法论＝memory `codex-art-spec-vs-paint`——其中**平台无关的部分继续有效**：描述词才干活（别引用旧图名）／负面点名具体病＝反向提醒／形状词＞萌形容词／情绪词控色／约束少而精／内容+画风写死、其余放开。Codex 专属操作条款（新对话/串行/无垫图参数）作废。
- 本地管线新增能力对旧方法论的两处修正：①"垫图"从此**真实存在**（垫图件直接吃参考图），不再纯靠文字；②同一性（跨视角/多姿态）优先走构图控制而非纯文字祈祷。
- **Qwen 用法铁律（L01-L05 五轮实证 · 2026-07-09 固化）**：
  - **Q1 禁令走负面口**：不要的东西写进独立"负面输入口"，正文只写"是什么"——正文里写 "no XX"＝反向提醒（老坑跨平台成立）。负面口对**物件级**禁令立竿见影（水面/火把/水花 L03 实证）。
  - **Q2 画风级母题换锚词治本**：万物给脸/吉祥物这类"它骨子里爱画的习惯"，负面口咬不动——真凶是锚词 **"chibi toy style" 词族**（L05 实证：换成 24批锚词"candy-cute cozy sci-fi + 软赛璐璐 + 糖果光泽"后 A 题脸从满屏降到偶发）；**chibi 只许修饰人**（驾驶员/居民），永不修饰载具与建筑。
  - **Q3 引子词黑名单（实证追加制）**："mobile game poster"→画手机壳；"port/港"→引海；"pirate boat"→水+火把；发现新引子**先删词**再谈禁。
  - **Q4 构图位占位法**：想赶走自发母题，用想要的东西把那个构图位写死占住（L04 实证：控制塔+信标天线挤走星星吉祥物）。
  - **Q5 劲度与兜底**：cfg 2.5 基线、3.2 可增强负面口不伤味；**多抽+挑永远是最后一道筛**；残余顽疾（偶发脸/泡舱人影）交改图手术（Qwen-Image-Edit）或画风小模型，不无限磨提示词（≤5 轮纪律）。
  - 在验条目：我方泡舱"只透操作台剪影"的稳定写法（40批 Ron 口径·敌方带小海盗=合规例外）。

## §4 验收守门 SOP

1. **Ron＝唯一审美裁判**；总控预筛只剔"明显崩坏/残缺/离题"，风格判断不代拍。
2. 单位类资产加两测：**35% 缩小三秒测**（并排小图 3 秒认出哪型哪艘）；**跨视角并排验收**（3/4 与俯视一眼认同一艘）。
3. 所有批次呈图时**与锁样并排**给 Ron（"是不是一家"一眼比）。
4. 定稿动作＝回流锁样定稿区＋锁样记录 md（结论/范围/配方/遗留四段）。
5. 接入后真机抽验帧率/发烫（配合工程线）；**AI 生成视频/连续帧绝不进包**（铁律A1）。

## §5 专属画风小模型（锁味的最终手段）

- 路线：L 批产图 → Ron 挑"是这个味"的入**教材库**（`F:\星舰小队本地出图\画风教材库\`，记来源批次）→ 攒够 ≥30 张起训（5090 本地训，小时级）→ **同题对比验收**（带/不带小模型各出一批，Ron 盲挑）→ Ron 认可后锁版本号记档 → 之后量产默认挂载。
- 防漂移：每次大批量产前先出 10 张体检对锁样；小模型升级需重新同题验收，旧版本保留可回滚。
