# 《我的星舰小队》资源货币方案 v2（首发版）

状态：首发版方向（Ron 拍板主体，部分细节待确认）。已选“保留单一抽卡”（原 v1 的方案 B），覆盖下方 v1 提案的 A/B 取舍。

## 货币清单（首发）
- **软货币（3，每场战斗必给；同时由基地离线产出—方案2）**：星矿（建基地）/ 星舰合金（仅星舰升级）/ 驾驶记录（驾驶员升级）。
- **进阶材料（2，主要抽卡：先得本体，重复转对应碎片=每单位专属）**：星舰碎片（星舰升阶，升阶顺带解锁插件槽/星核槽）/ 驾驶员碎片（驾驶员升星）。补充定向来源：行动宝库（指定）、7 天过程、商店。
- **星核体系（2）**：星核碎片（随机合成）/ 星空宝石（星空宝库定向兑换，含限定核）。来源：精英/Boss、打捞（稀有/史诗信标）、商店、3 天活动、星辉货舱、7 天完成。
- **商店货币（1）**：星贝（商人小站；回收多余物品获得 + 买中端货）。
- **打捞信标（3，加法分层）**：普通=3 软货币；稀有=普通+星核碎片；史诗=稀有+星空宝石。来源：商店购买、星辉货舱、关卡概率（Boss 必史诗；精英必稀有+小概率史诗；普通 中小概率普通/小概率稀有/极低史诗）。
- **抽卡（方案 B）**：单一「补给券」+ 抽取选池（招募=驾驶员 / 整备=星舰）。

## 砍掉
- ❌ 太空补给（软货币已每战直给）；❌ 改装图纸（槽位随升阶解锁）；❌ 星辉信标、奇迹信标；❌ 招募券/整备券（并入单一补给券）。

## 插件：只分品质、不分等级（建议采纳）
- 不做插件升级/升级货币（故星舰合金仅供星舰）。
- 理由：①轻量；②升级会让玩家舍不得换、破坏“自由换插件试搭配”乐趣，品质制下零负担实验；③进阶靠“用更高品质替换”。
- 深度来自“插件效果多样性”，非数值成长；高品质=更强数值（传奇可带一点额外效果）。

## 活动奖励（首发 + 建议优化）
- 3 天行动：过程=偏信标/碎片/补给券（少给每战已有的软货币）；完成=星辉货舱；结算=行动宝库（传奇插件 / 指定星舰碎片 / 指定驾驶员碎片 三选一）。
- 7 天扩张：过程=基础资源/居民/工人/稀有信标/碎片；完成=史诗信标/星核碎片；结算=扩张宝库（1 次自选核，之后随机三选一；留广告接口本轮不做核广告收益）。
- 星辉货舱：随机量 星核碎片+星空宝石 + 3~5 随机品质信标。
- 保留“跨期追赶”，忙完回来不亏。

## 待确认 / 风险
- 🔴 **补给券来源未定**：纯广告无内购，补给券须靠玩法稳定产出（主线/Boss首通/活动/打捞/商人）；建议加回“赞助补给（看广告→补给券）”作为变现点。节奏表必须验证免费玩家够抽。
- 🟡 居民/工人不应只靠 7 天给：基地“变热闹”是核心爽点，需更频繁日常来源。
- 🟡 活动过程奖励应偏稀有物（信标/碎片/券），别重复每战软货币。
- 🟡 首个行动宝库的传奇插件在 ~D3 可能偏强：注意 pacing（或给更温和助力/新手向传奇）。
- 🟡 星核碎片/宝石来源广：节奏表用合成成本/兑换价压回核获取节奏（碎片≈每5-7天、宝石≈每10-14天）。
- 🟡 升阶改 per-unit 碎片（覆盖原“无每船专属碎片”）：确保定向碎片来源足，稀有单位也能升阶。

# （历史）资源货币方案（提案 v1）

状态：提案 / 参考，待 Ron 确认。目标：在不破坏任何已锁定玩法的前提下，把资源货币精简到最少、各司其职、贴合“轻量 + 良心”定位。

## 一、设计原则
1. 每种货币只负责一条清晰的“消耗赛道”，职能不重叠。
2. 总是一起赚、一起花的两种材料 → 合并。
3. “箱子”是奖励包装，不算玩家要记忆的货币。
4. 最大爽点（星核）的体系单列、最重；软货币尽量少且直觉。

## 二、提案货币清单（分层）

### 第 1 层 · 养成软货币（稳定流入，数值上涨的燃料）——3 种
| 货币 | 用途 | 记忆点 |
|---|---|---|
| 星矿 | 基地建设与升级 | 盖房子 |
| 合金 | 星舰升级 + 插件升级 | 强化战斗硬件 |
| 战斗记录 | 驾驶员升级 | 练人 |

### 第 2 层 · 进阶材料（控大跳，较稀）——2 种
| 货币 | 用途 | 备注 |
|---|---|---|
| 星舰图纸 | 星舰升阶（按品质/类型通用） | **升阶顺带解锁插件槽/星核槽** → 取消“改装图纸” |
| 驾驶员碎片 | 驾驶员解锁 / 升星 | 通用碎片，避免每人专属碎片 |

### 第 3 层 · 星核体系（核心爽点，双渠道，已锁定）——2 种
| 货币 | 用途 | 性格 |
|---|---|---|
| 星核碎片 | 随机合成星核 | 赌、惊喜 |
| 星空宝石 | 星空宝库定向兑换星核 / 专属星舰 | 定向兜底、限定核独占 |

### 第 4 层 · 商店货币——1 种
| 货币 | 用途 | 备注 |
|---|---|---|
| 星贝 | 商人小站（回收多余物品获得；买信标/图纸/碎片/星辉货舱等中端货） | 同时是“多余物品回收”的去处（item sink） |

### 第 5 层 · 打捞入场券（奖励池分层）——精简到 3 种
| 信标 | 保底方向 |
|---|---|
| 普通信标 | 基础资源 |
| 稀有信标 | 稀有资源（并入原“星辉档”高端：星辉货舱、图纸等） |
| 星核信标 | 星核碎片 / 星空宝石 |

（取消 星辉信标、奇迹信标）

### 奖励包装（非货币，不计入记忆负担）
太空补给（普通箱）、星辉货舱（三选一 + 广告再选一）、行动宝库（3 天，含首个 Boss 助力）、扩张宝库（7 天，主打星核）。

## 三、砍掉 / 合并了什么（及原因）
- ❌ **改装图纸** → 并入星舰升阶。升阶到指定阶自动解锁插件槽 1/2/3 与星核槽：少一种货币，且让升阶更有获得感。
- ❌ **奇迹信标** → 取消。奇迹建筑已简化为只留星核展厅，由主线里程碑解锁、升级用星矿 + 少量指定材料。
- ❌ **星辉信标** → 并入稀有信标高端，少一档信标。
- ❓ **补给券 / 招募券 / 整备券** → 取决于“是否保留星港补给站抽卡”，见第四节。

## 四、关键决策点：要不要抽卡（星港补给站）？
本作最大爽点（星核）不在抽卡池；抽卡池只产驾驶员/星舰/插件这类非核心层，且已有大量其他渠道。两个干净的走法：

**方案 A（推荐，首发）：无抽卡。**
- 取消 星港补给站 + 补给券 + 招募券 + 整备券 + 赞助补给。
- 驾驶员：主线救援（救回居民/工作人员，直接给整个驾驶员，超贴主题）+ 驾驶员碎片解锁/升星。
- 星舰：星舰图纸 + 船坞建造（“你在修星港、造船”，主题自洽）。
- “开箱惊喜”由星辉货舱 / 宝库三选一承载（拿到就开，不做老虎机）。
- 优点：最轻、最良心、最省货币；缺点：少一个“攒券抽一发”的即时赌点。

**方案 B：保留单一抽卡。**
- 只留一种“补给券”+ 抽取时选池（招募/整备），合并掉三种票。
- 优点：保留经典抽卡爽点（也利于后期内购）；缺点：多一套系统 + 一种货币。

> 建议：首发纯广告 + 轻量 + 良心 → 走 A；后期上内购真需要经典抽卡时再加回 B。

## 五、提案后玩家需记忆的货币数
- 方案 A：星矿/合金/战斗记录 + 星舰图纸/驾驶员碎片 + 星核碎片/星空宝石 + 星贝 + 3 信标 = **10 种**。
- 方案 B：+ 补给券 = 11 种。
- （原方案约 18–21 项：~13 货币 + 5 信标 + 3 票。）
