## D0 · 通用机制与口径（定义一次，各条引用）

- **战场格子**：我方 **3×3（9 格）**；敌方 **5 行 × 7 列（35 格，可配置）**。范围一律用"格"说清。
- **"相邻"两种**：**相邻 4 格** = 上下左右（不含斜角）；**相邻 8 格** = 含斜角的一圈。
- **等级上限**：星舰 = 船坞等级 ×10；驾驶员 = 训练舱等级 ×10（建筑满 lv10 → 满级 **100**）。前期实际练到 ~35 级（D21，对齐 B1）。
- **升级 vs 升阶/升星（核心分工，回答"升级有什么意义"）**：
  - **升级 = 稳定的数值骨架**（撑起战力墙的数学、保证"每天养成都变强"）**+ 前期等级里程碑解锁基础技能**（教学期正反馈）。
  - **升阶/升星 = 阶梯式质变**（开槽、解锁高级技能、能力质变）。
  - **真正的"千变万化"靠四层搭配（驾驶员/插件/星核），不靠堆等级**——这是有意分工。
- **星舰升阶（所有星舰统一，不再每艘写）**：C=1 插件槽 / B=2 槽 / A=3 槽 / **S=开星核槽** / SS=技能终极强化。花该舰专属碎片（20/40/80/160）。各舰只写"自己特有的升阶解锁"。
- **驾驶员系数（B1，升星用）**：1★1.00 / 2★1.08 / 3★1.18 / 4★1.30 / 5★1.45。
- **"重击"定义**：单次受到伤害 ≥ 本舰最大生命 **15%**（暂定）。
- 下文数字 = **暂定·符合 B1 节奏、待真机调**。

## D1 · 星舰 ×2

### 星舰 1：赤焰（火炮型 / 自由型，新手第一艘）
- **外形**：圆润又结实的中型炮艇，赤红涂装、一对粗短主炮，憨里带凶。
- **属性**（C 阶 Lv1，暂定）：HP 800 / 攻击 60 / 攻速 1.0 次每秒 / 防御 20。
- **普攻**：朝**最前排 1 个敌人（1 格）**打一发直射炮，攻击 ×1.0。
- **技能**：
  - 「集火炮」（CD 技，每 8 秒）：对**当前目标 1 个敌人（1 格）**造成 攻击 ×3.0。（打谁由驾驶员定，默认最前排。）
  - 「快速装填」（被动）：普攻 20% 概率立刻再打一发。
- **升级**（花星舰合金）：每级 HP/攻击 ×(1+0.08×(级-1))。**等级里程碑（解锁基础技能）**：Lv1 只有普攻 → **Lv5 解锁「集火炮」** → **Lv15 解锁「快速装填」** → 之后纯数值成长。
- **特有升阶解锁**：B → 集火炮命中后追加一发（攻击 ×1.5）；A → 集火炮伤害 ×3.5；SS → 集火炮命中"目标 + 同列上下各 1 格（共 3 格）"。
- **视觉关键词**：赤红、炮口余焰、稳重憨实。

### 星舰 2：磐石（壁垒型 / 专属型，绑定驾驶员「铁岩」）
- **外形**：厚重的六边形盾舰，正面一面能展开的蓝色大能量盾。
- **属性**（C 阶 Lv1，暂定）：HP 1500 / 攻击 30 / 攻速 0.7 / 防御 40。
- **普攻**：**射出一片光盾，打最前排 1 个敌人（1 格）**，攻击 ×1.0。（远程投射、不移动）
- **技能**：
  - 「护盾力场」（被动）：自己 + **相邻 4 格**友军，受到伤害 **−15%**。
  - 「张盾」（CD 技，每 12 秒，持续 4 秒）：期间 自己 + 相邻 4 格友军 受伤 **−40%**。**【铁岩驾驶时：范围扩大到相邻 8 格（含斜角），减伤 −55%】**＝专属发挥。
- **升级**：每级长 HP/防御。里程碑：Lv1 普攻 → **Lv5 解锁「护盾力场」** → **Lv15 解锁「张盾」** → 之后纯数值。
- **特有升阶解锁**：B → 张盾持续 5 秒；A → 张盾减伤 −50%；SS → 张盾期间被打中时反弹 20% 伤害。
- **视觉关键词**：厚甲、蓝色能量盾、稳如山。

## D2 · 驾驶员 ×2

> 驾驶员**不自带技能、不加血加攻**。成长拆成两条独立的线：**升级=平滑变强（天天可升）**、**升星=阶梯质变（偶尔一大跳）**。

### 驾驶员 1：赫炎
- **介绍**：炮手型驾驶员，年轻气盛的天才神枪手——专管"把火力堆到最猛、一个个收掉残血"。
- **外形**：Q 版青年，利落短发、笔挺飞行夹克、自信坏笑，神枪手气质。
- **驾驶能力**：让本舰**优先集火"血量最少的存活敌人"**（实现 = 改写本舰目标选择）。
- **驾驶天赋「过热」**：连续命中层层叠加伤害，每层 +X%，最多 5 层；3 秒内没再命中则清空。
- **【升级】Lv1→100（花驾驶记录）**：
  - 逐级：装该员的星舰 **+3 战力**；「过热」每层加伤 **+0.05%**（≈每 10 级 +0.5%）。
  - **等级大节点**：**Lv30 / 60 / 90 → 过热叠层上限 +1**（5→6→7→8 层）。
- **【升星】1★→5★（花专属碎片 20/40/80/160）**：
  - 逐星：驾驶员系数跳档（1★1.00→…→5★1.45，**战力大跳**）+「过热」每层**基础档抬升**（4%→5%→6%→7%→8%）。
  - **质变大节点**：**3★ →** 击杀敌人时缩本舰技能 CD 1 秒；**5★ →** 过热满层时下一发普攻**必暴击 + 溅射到相邻敌人**。
- **过热每层加伤示例（基础档 + 等级微调）**：1★·Lv1 = 4%（满层 +20%）→ 3★·Lv30 = 7.5%（+37.5%）→ 5★·Lv100 = 13%（+65%）。

### 驾驶员 2：铁岩
- **介绍**：护盾型驾驶员，磐石的专属驾驶——沉默可靠的老兵，护住队友是本能。
- **外形**：Q 版方脸壮汉，肩扛半截旧盾，脸上一道疤、憨笑。
- **驾驶能力「光盾守护」（替代原"常驻嘲讽"）**：**当一名后排队友即将挨打时，磐石射出光盾替它挡下这一次伤害**——这次伤害改打在磐石身上、被磐石的高防/减伤扛掉；**冷却 2 秒（每 2 秒最多挡 1 次）**。
  > 实现 = "伤害拦截/转移"积木：监听落向后排队友的攻击，若磐石存活且守护就绪 → 把这次伤害转到磐石。保留了"打后排"的博弈（不像常驻嘲讽把所有火力一锅端），又让铁岩"该救谁救谁"。
- **驾驶天赋「反震」**：被击中时格挡掉一部分伤害，并把"格挡掉的量 ×反弹比例"弹回攻击者。
- **【升级】Lv1→100**：逐级 +3 战力 +「反震」格挡 +0.05%/级；**等级大节点 Lv30/60/90 → 反弹比例 40%→50%→60%→70%**。
- **【升星】1★→5★**：逐星 系数跳档 +「反震」格挡基础档抬升（10%→14%→18%→22%→26%）；**质变大节点：3★ →** 守护冷却 2 秒→1.5 秒；**5★ →** 自身血量 <30% 时自动张一次护盾（吸收 = 最大生命 15%，每场 1 次）。
- **格挡比例示例**：1★·Lv1 = 10% → 3★·Lv30 ≈ 19.5% → 5★·Lv100 ≈ 31%。

## D3 · 星核 ×2（按放置军团宝物规律：一句话看懂 + 强力 + 改写规则触发连锁/克制）

> （固定新手核「过载核心」另定，首个 Boss 首杀人人必得，不在本批。）

### 星核 1：银河烟花（通用 · 连锁清群）
- **质变（大白话）**：每当你**打死一个敌人，本舰立刻自动朝另一个敌人补一发**；如果这一发**又把敌人打死，就再自动补下一发**……像多米诺骨牌一样**一连串打下去（最多连 5 发）**，而且越往后伤害越高（每多一发 +10%）。（这种"一杀引发下一杀"的串联 = **连锁**。）
- **为什么强 / 协同**：配赫炎（专杀残血）→ **一杀引爆雪崩式连锁**，瞬间扫空一排小怪。
- **视觉**：每段连锁拉出一道彩色弹道，像放烟花串。
- **类型 / 获取**：通用型 / 扩张宝藏三选一 · 碎片合成 · 宝库兑换。

### 星核 2：黑洞（场景 · 专克肉与 Boss）
- **质变**：本舰**技能每次命中，额外造成"目标当前生命值 ×8%"的伤害**（对 Boss 也有效；单次上限 = 本舰攻击 ×5，防止失衡）。
- **为什么强 / 协同**：敌人越肉、Boss 血越厚越值钱（学神银弩箭按%扣血）；配「弦」易伤、「冷却芯片」高频技能 → 稳定碾血墙。
- **视觉**：命中处浮现小紫黑漩涡，"吸"掉一块血。
- **类型 / 获取**：场景型 / 扩张宝藏三选一 · 碎片合成 · 宝库兑换。

## D4 · 插件 ×6（每个一个名字，三档品质都给暂定值）

> 同一插件分**精良/优秀/传奇**三档；变强靠"换更高品质 / 合成升品"。传奇 = 强数值 **+ 一条额外小效果**。
> **战力贡献**（B1 统一折算）：精良 15 / 优秀 35 / 传奇 70；下表是**实际作用效果**（暂定）。

### 武器槽
**① 增幅核心**　精良 +8% 伤害 / 优秀 +18% / 传奇 +35% **+ 暴击时把该次 30% 伤害溅射到相邻 1 格敌人**
**② 瞄准镜**　精良 +8% 暴击率 / 优秀 +18% / 传奇 +30% **+ 暴击时额外补一下（攻击 ×0.5）**

### 技能（CD）槽
**③ 冷却芯片**　精良 −8% CD / 优秀 −15% / 传奇 −25% **+ 开局第一个技能 CD 直接减半**
**④ 增幅线圈**　精良 +10% 技能伤害 / 优秀 +22% / 传奇 +40% **+ 技能作用范围 +1 格**

### 战术槽
**⑤ 护盾发生器**　精良 −8% 受伤 / 优秀 −15% / 传奇 −25% **+ 受到"重击"时给自己罩护盾（吸收 = 最大生命 10%，每场 3 次）**
**⑥ 破甲弹头**　精良 +15% 破盾效率 / 优秀 +30% / 传奇 +50% **+ 破盾时让该敌人防御 −20% 持续 3 秒**
