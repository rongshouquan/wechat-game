## 二、按玩法看：每个玩法的“主要产出（玩它的理由）”+ 次要产出

### 1. 主线推关（普通 / 精英 / Boss，4 星域）— 理由：推进、解锁一切、永久抬高离线基线
- **主要**：软货币（每战必给，大头）· 首通解锁（救回驾驶员本体 / 建筑 / 系统）· 补给券（关卡首通、Boss 首通）· 精良插件（关卡掉落；精英/Boss 小概率优秀）· 永久抬高离线产出基线 · 居民（推进救回居民，基地变热闹）· 首颗星核（第一个 Boss 的“星核宝藏”三选一）
- **次要**：重复挑战极少基础资源、小概率普通信标；精英/Boss 小概率星核碎片
- 分工：普通关=爽推+软货币+精良插件（小概率普通/稀有信标）；精英关=**必得稀有信标**+机制考验（小概率史诗）；Boss 关=**必得史诗信标**+补给券+里程碑
- 注：**里程碑** = 达成进度节点的一次性奖励（首通 / 累计达标 / 解锁），非独立系统。

### 2. 基地建设养成（含人口）— 理由：变强（全队乘区）+ 变热闹（成就感）+ 抬高离线上限
- **本质是 SINK + 乘区**：吃星矿/材料，产出“永久加成与容量”——全队属性加成（研究塔）、离线储存上限（居住舱）、打捞队数（星港）、降基建消耗（工人）。
- 不直接产资源，产的是“倍率”和“容量”。

### 3. 离线产出（基地被动，方案2）— 理由：挂机收入骨架
- **主要**：软货币（星矿/合金/记录）的稳定大头；速率随“已通关最高星域”提升，上限由居住舱决定。
- 次要：无。（跨期追赶核心，见第四节。）

### 4. 打捞（信标 → 派队 → 定时返还）— 理由：稀有物的定时滴流
- **主要**（由信标档决定奖励池）：普通信标→软货币；稀有信标→+星核碎片、通用碎片、插件；史诗信标→+星空宝石。
- 次要：软货币、补给券小概率。
- 广告点：看广告缩短打捞剩余时间。

### 5. 抽卡（星港补给站）+ 赞助补给 — 理由：抽到新驾驶员/星舰（收集与编队广度）
- **主要**：驾驶员本体（招募池）/ 星舰本体（整备池）；重复→对应专属碎片。
- **两池只出本体（不出插件）**：招募=驾驶员、整备=星舰。插件来源另有 5 条（关卡/打捞/商人/合成/行动宝库），不进抽卡。
- 货币：补给券。**赞助补给 = 看广告得补给券**（首发纯广告的核心变现点；每日上限，详见广告设计）。

### 6. 商人小站（回收 → 星贝 → 买）— 理由：把多余东西换成想要的中端货
- **主要**（花星贝）：信标 / 通用碎片 / 插件 / 补给券（少量）/ 星核碎片（限量）。
- item sink：回收多余低品插件、多余信标、指定材料 → 星贝。
- 防套利：周期购买上限、回收折损、不可回收表（细表定）。

### 7. 星空宝库（花星空宝石）— 理由：定向兜底拿到想要的核 / 限定核 / 专属星舰
- **主要**：星核（含宝库限定核）/ 专属星舰；一次性解锁（买过显示已拥有）。

### 8. 3 天行动 — 理由：装备你的船（插件）+ 升阶你想要的单位
- 过程（偏稀有物、少软货币）：信标 / 补给券 / 通用碎片。
- 完成：星辉货舱。
- 结算：行动宝库（**传奇插件 / 指定通用星舰碎片 / 指定通用驾驶员碎片 三选一**）。
- 主要产出定位：**插件（传奇）+ 定向碎片**。

### 9. 7 天扩张 — 理由：拿星核（质变）+ 扩充基地人口
- 过程：更高量基础资源 / 居民 / 工人 / 稀有信标 / 通用碎片。
- 完成：史诗信标 / 星核碎片。
- 结算：扩张宝库（第 1 次自选完整星核；第 2 次起随机星核三选一；留广告接口，本轮不做核广告收益）。**【不改】**
- 主要产出定位：**星核 + 居民/工人 + 史诗信标**。

## 四、跨期追赶（小幅折损，曲线匹配 Ron 口径）

目标曲线：**少玩 1-2 天 → 追回差不多 / 差一点点；少玩 3-4 天 → 明显比天天玩的慢。** 既不让“少玩几天落后很多”，也不让“少玩几天无所谓反正追得上”。

机制（多为“上限 + 窗口”自然形成，不额外加系统）：

1. **离线产出上限 = 天然追赶曲线（核心）**：离线攒满即停、溢出浪费。把上限设在 **≈ 1.5 天产出**（居住舱可延长）：
   - 少玩 1-2 天：上线一次领回接近满上限的一大笔 → **追回大部分，仅差一点点**（略有溢出损失）。这一刻“领一大笔”是爽点，不是惩罚。
   - 少玩 3-4 天：早就攒满、后续产出全溢出浪费 → **明显落后**（白损失 2-3 天产出）。
2. **3天行动 / 7天扩张 = 窗口追赶**：周期内任务可随时补做。少玩 1-2 天，剩余时间够补完拿到宝库；少玩 3-4 天，时间不够、可能丢该轮结算宝库。
3. **打捞 = 错过即损失新循环**：队伍回来待命不浪费“已完成那次”，但不自动出新的。少玩多天 = 少跑多趟（线性小损失）。
4. **不做“假期全额补偿”**：故意不补满缺勤时间，确保天天玩的人始终领先。

> 北极星兑现（体验平滑不憋屈）：少玩 1-2 天靠“上线领一大笔 + 补完活动窗口”平滑追回，不憋屈；少玩更久则真实落后，尊重每日玩家。上限/窗口的具体数值由节奏表锁定。

## 五、商人小站详细设计（可卖 / 可买 / 稀有刷新）

货币：星贝（回收多余物品获得）。定位：item sink + 中端定向购买 + “逛店惊喜”。

### 可卖（回收 → 星贝，有折损）
- 多余 / 重复插件（含传奇重复）。
- 多余普通信标、指定杂项材料。
- 回收价 < 潜在价值（折损），避免“回收—购买”套利。
- **不可回收表**：本体、星核、专属星舰、星核碎片、星空宝石、各类碎片、补给券（保护核心进度货币）。

### 可买（花星贝，固定商品格 + 周期刷新）
- 信标（普通 / 稀有；史诗只在稀有刷新里偶现）。
- 通用碎片（舰 / 员）。
- 插件（精良 / 优秀）。
- 补给券（少量，限购）。
- 星核碎片（限量限购）。

### 稀有刷新（“低概率刷出贵的好东西”）
- 除固定格外设少量“随机格”，每周期刷新时**低概率刷出稀有高价商品**：传奇插件 / 史诗信标 / 较多通用碎片 / 较多星核碎片 / （极低概率）少量星空宝石。
- 标价高（要攒星贝）、限购 1 次、刷到才有 → 制造“逛商店的期待感”，给“攒星贝”一个目标。把星核的“赌 + 熬 + 惊喜”逻辑迁到日常商店。
- 养“商人”（人口 / 商人建筑等级）→ 提高稀有刷出概率与刷新质量 → 给养成商人明确正反馈。

### 商人小站升级效果
- 回收得更多星贝、商品格增加、高品出现率↑、稀有刷新质量↑、可手动刷新次数↑（看广告或星贝）。

### 防套利（细表定数值）
- 回收折损、周期购买上限、刷新上限、不可回收表、星贝不能无限循环刷资源。

> 北极星兑现：每周期一次“逛店看有没有刷到好东西”的轻惊喜（主动、愉悦，不是日常任务式压力）；多余物品永远能换星贝，无死物品。
