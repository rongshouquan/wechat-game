## 0. 开场（进游戏）
- 交代背景：你是新上任的**星港指挥官**，接手一座破败小星港；星域又乱又热闹（星盗 / 失控无人舰 / 废铁机械 / 星能污染舰）。
- 你要做什么：**收集星舰、组建小队、打败敌人**。
- 目的：把小破星港**养成热闹大星港**。

## 第 1 关 · 纯普攻 + 首次三选一（教"自动打" + 资源进账）
- **战场**：我方 = **极焰号（C 级 Lv0）+ 炎（1★ Lv0）**；敌方 = 一前一后 2 个，前者高血低攻、后者低血高攻。
- 引导点「开始战斗」→ 观看自动战斗：**只有普攻**（炎 Lv0 尚无驾驶能力），极焰号默认**先打最近（前）、再打远（后）**；打赢、自身**剩约 1/3 血**（让玩家觉得有点险）。
- **胜利必得**：星舰合金 + 驾驶记录 + 星贝。
- **三选一**：武器槽精良插件 / 普通信标 / **星矿** —— 教学**强制引导选星矿（+50）**（下一步解锁建筑要用）。

## 第 1 关后 · 解锁建筑 + 升级（舰 & 员 都要练）
- 引导点「返回星港」→ 主界面**所有建筑未解锁**。
- 引导用刚得星矿**解锁 Lv1 船坞 + Lv1 驾驶员训练舱**。
- 进**船坞**：**极焰号升到 Lv1 → 解锁星舰技能「集火炮」**（展示介绍）。
- 进**训练舱**：**炎升到 Lv1 → 解锁驾驶能力「集火血量最少的敌人」**（展示介绍）。
- 引导点主界面「出战」→ 进第 2 关备战。

## 第 2 关 · 展示"技能 + 驾驶能力" + 装插件（教战力提升 & 插件）
- **备战界面**：敌人比上关多 1 个（前排 1 高血低攻、后排 2 低血高攻）。
- 点开始 → 演出：极焰号（炎驾驶）**开局技能「集火炮」秒掉 1 个低血高攻** → 普攻另一个低血高攻 → 再打前排高血低攻 → 最后一只残血时集火炮 CD 转好、再放一发收掉。**敌 3 个，打完极焰号还剩半血**（明显比上关轻松 = 战力提升感）。
  > 同时展示了**星舰技能（集火炮）+ 驾驶能力（炎专挑低血打）**。
- **三选一**：强制选**武器槽精良插件** → 弹窗「恭喜获得 **急速弹链**（武器槽·精良）」；点图标可看效果；下方仅「确定」→ 引导返回船坞装插件。
  > （变更#4：三选一里只显示"武器槽·精良"，选后才揭晓是"急速弹链"。）

## 第 2 关后 · 插件装配 + 槽位教学
- 船坞装配界面：**只 1 个槽开着、另 2 个锁着** → 介绍：另两个是**技能槽 / 战术槽，分别在星舰升到 B 阶 / A 阶自动解锁**。
- 装上急速弹链 → 引导打第 3 关。

## 第 2 关后续 · 认识 3 天行动 / 7 天扩张（早期 · 短教程）
- 回基地时引导点开**3 天行动 / 7 天扩张**入口 → 🟦 短教程：「这里是限时活动——**正常推关 / 升级 / 打捞 / 抽卡就会自然累积进度**（不用专门打卡），到段位领过程奖励、完成领宝藏（**7 天扩张能拿完整星核**）。」
- 让玩家先点进去看一眼、知道有这回事；**不强求现在做完**（平时玩就会推进度）。放在这里是为了第一天就让玩家心里有"这两个活动在攒奖励"。

## 第 3 关 · 新敌人 + 抽卡扩编队（教抽卡 & 第二队伍来源）
- **战斗**：对方上场**一种新敌人类型**，靠新插件加成**勉强打过**。
- **三选一**：强制选**补给券（+2）**。提示玩家"敌人越来越强，得赶快扩充编队"。
- 引导回星港**激活星港补给站** → 进去**驾驶员池 / 星舰池各抽 1 抽** → 获得 **C 级铁壁号（护卫）+ 1★ 砺（护卫驾驶）**。
- 引导把**两艘星舰 + 两名驾驶员都升到 Lv5** → 引导继续主线。

## 第 4 关 · 上阵新舰 + 装配驾驶员 + 摆阵 + 打捞
- **备战界面**：敌人又变多。
- 教**上阵铁壁号** + 铁律「**星舰必须配驾驶员才能上场**」。
- 教在**备战界面直接给星舰装配驾驶员**（说明：装配**不必回星港**，**只有升级 / 升阶才必须回星港**）。
- 上场后讲：**铁壁号 = 护卫舰（站前排抗伤）**、**极焰号 = 突击舰（站后排有良好输出环境）** → 引导**拖铁壁号到第 1 排、极焰号到第 2 排**（摆阵 / 九宫格换位教学）。
- 点开始 → 因新舰 + 都 Lv5 + 站位合理，**这关虽敌更多更强、却几乎无损轻松赢**。
- **三选一**：强制选**普通信标** → 引导**解锁打捞港** → 选刚得的普通信标 → 选 **2 小时** → 开始打捞。

## 第 5 关 · 克制墙 + 商人/补给券（需求驱动 · 强引导）
- **敌情**：后排出现一个高威胁**炮台**（铁壁挡前、但它在后排一直点我方极焰），现有搭配（炎打最低血 / 砺嘲讽）够不到后排、卡住。
- 引导：「后排那门炮很烦，得有人专打后排」→ 引导**抽卡补强** → **补给券不够** → 引导去**商人小站用星贝买补给券**（教商人 + 补给券）→ 抽卡 → 脚本给到 **「影」（突击驾驶·专打最后排）** → 把影装到极焰号 → 极焰号专点后排炮台 → 过关。
  > 教学：克制（换驾驶员应对后排）+ 商人小站 + 补给券购买 + 再次抽卡（顺带凑编队）。

## 第 5 关后 · 首次升阶 + 首次升星（养成深化 · 短教程）
- 攒的专属碎片够了 → 引导**第一次升阶**（极焰号 C→B）：
  - 🟦 **首触短教程（1-2 弹窗·可跳过/下一步）**：「升阶 = 花该单位**专属碎片**提升阶级 C→B→A→S→SS；每阶**开新插件槽 / 强化技能**，**A 阶、SS 阶有大质变**；碎片主要从抽卡来。」
  - 升 B → **开第 2 插件槽（技能槽）**（呼应第 2 关埋的伏笔）→ 引导装一个技能槽插件。
- 引导**第一次升星**（炎 1★→2★）：
  - 🟦 **首触短教程**：「升星 = 花**驾驶员专属碎片**提升星级、**强化驾驶能力**，**3★、5★有大质变**。」
- **养出"第一艘 S 阶"接陨星弹（不强制哪艘·玩家自选）**：强引导这里带玩家走几次升阶（首触教程把"升阶"练熟）；**关键靠资源投放节奏保证**——玩家正常玩到"能打首个 Boss"那一刻，手上累积的专属碎片/合金**刚好够把任意一艘星舰养到 S 阶（哪艘自己选）**，于是首 Boss 掉陨星弹时已有一艘 S 阶船开了星核槽、能合法接住。不逼玩家练极焰号。

## 穿插（不锁时机 · 首次触发即短说明）
- **离线收益**：玩家首次离开再回来 → 自动弹**离线收益领取** + 🟦短说明「挂机也在产资源、上线一键领」。（若一直连打没离线，新手期某节点强制演示一次。）
- **居住舱**：主线救回居民时 → 引导点**居住舱** + 🟦短说明「居民 → 离线收益↑ + 存储时间↑；工人 → 建筑升级材料↓」。

## 第 3 天 · 首个 Boss · 星核（强引导收尾）
- 推进到**首个 Boss「海盗大副」**（**n030 · 第5章末**，成就门控·约 D3，2026-07-03 对齐附录D 敌人真源 §5 口径）→ 打赢 → 掉 **陨星弹（固定新手核）**：
  - 🟦 **首触短教程**：「**星核 = 本作最稀有的宝物**，装上直接**质变战斗**！这颗陨星弹是首杀 Boss 的奖励——普攻变原子炮、开局一炮轰半场。」
  - 引导去船坞**把陨星弹装上（升到 S 阶、开了星核槽的那艘）**感受质变。〔**星核展厅**（＝碎片合成/宝石兑换入口）**已早期可用·与首Boss解耦**——不在此步"解锁"；精确解锁时机（开局花资源 / 第2-3天 / 关10-15-20 提示）留第三块数值校准定稿，见 GDD S7 建筑解锁节奏。Ron 2026-07-04 拍板。〕
  - 此时玩家手上已有**一艘自选的 S 阶星舰（开了星核槽）** → 陨星弹**直接装上（合法、无需特例）**，新人第一次就强烈感受"核=变强很多"。（怎么保证打到这有 S 阶船：见上节"资源投放保证够养 1 艘 S 阶"。）
- 至此**核心循环全部教完**（四层战斗 / 升级·升阶·升星 / 插件 / 抽卡碎片化 / 装配 / 摆阵 / 克制 / 打捞 / 商人补给券 / 离线 / 居住舱 / 星核）→ **强引导结束，放开让玩家自由探索养成搭配、挑战更高关卡**。

## 弱引导 ＝ "首次接触即短教程" 系统（强引导结束后 · 长期）
> **规则**：任何**第一次接触**的系统/内容 → 触发 **1~3 个短弹窗教程**，**可"跳过 / 下一步"、只首次触发、以高亮提示为主、不锁死操作**。（区别于关 1-4 强引导＝手把手锁操作、不可跳。）
- **首触清单（示例·各自第一次触发时弹）**：3 天行动 / 7 天扩张 / 插件合成升品 / 背包碎片合成新单位 / 研究塔 / 星空宝库（星空宝石兑换）/ 邮件领取 / 打捞带回稀有 / 商人稀有刷新 / 跨期追赶 / 打捞加速券 / 7 天扩张自选星核装配 等。

## 设计自检（深度 · Claude）
- ✅ **节奏**：强引导 关1 → D3 首核 覆盖全部核心循环，之后放开；不拖沓、不缺漏。
- ✅ **自然过渡**：每个系统由"需求/剧情"带出（卡后排→抽卡→商人；救人→居住舱；Boss→星核），非干巴列清单。
- ✅ **尽快放开**：核心一教完（D3）就交还玩家自由摸索养成搭配 / 挑战更高关。
- ✅ **首次接触都有短教程**：升阶/升星/星核/活动等首触都有 1~3 弹窗、可跳过/下一步。
- ✅ **平滑不憋屈**：关 5 克制墙＝换搭配就过（脚本保证抽到克制单位），不真卡死。
- ✅ **强/弱引导分明**：关1-4 强引导锁操作不可跳（教核心必走）；首核后弱引导短教程可跳。
- ✅ **首Boss 装核（已按 Ron 定·不用特例）**：强引导持续喂极焰号专属碎片、引导它在首 Boss 前升到 S 阶、开星核槽，陨星弹合法装上。**v1.0/设计决策记录曾设想的"借核演示"（D1 临时借核试用当缓冲）已作废、不采用**（2026-07-03 显式声明，与本条"不用特例"一致）——首Boss 门槛达成保证改用下方"资源投放节奏"确保真实拥有 S 阶，不靠临时借用。
- ⚠️ **数值阶段必须校准的（资源投放节奏）**：把"能打过首个 Boss"的门槛 ≈ 卡在"玩家正常玩累积的资源**刚好够养出 1 艘 S 阶星舰**（已拥有本体只需三次升阶 50+100+300=**450** 专属碎片·暂定，哪艘自选；2026-07-03 按附录D 星舰真源 §0 新读法核对——30 是"背包碎片合成本体"的花费，起手船已拥有本体不花这笔，故不计入）"。预期分层：**快玩家**（全看广告+每天几小时+会规划）首 Boss 时可能养出 **1-2 艘 S 阶**；**普通休闲**（不看广告、打得慢）**晚几天**才打过首 Boss，但那时也凭已有资源够养出**第 1 艘 S 阶**。→ 首 Boss 后难度按"玩家手上有 1 艘强旗舰"来配。
- 📌 小提醒：① 满编 5 舰靠 关3抽(铁壁/砺)+关5抽(影)+主线救回 凑，脚本要确保 D1-2 凑到 4-5 套；② 离线收益首触依赖玩家真离线，必要时新手期强制演示一次。

> 📌 整理关1-4 时修正的一处：原"第1关胜利获得**星矿**"与变更#3 冲突 → 已改为**必得=合金/记录/星贝**，星矿走第1关三选一（强制选）。
