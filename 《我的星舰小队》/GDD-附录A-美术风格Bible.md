## A0 · 总调性与北极星视觉转译

- **一句话画风（=全局唯一锁样）**：**糖果软萌温馨 · 软赛璐璐 + 糖果光泽**（第15批版B-抽3 · Ron 2026-07-05 锁）。明亮、暖、透亮的糖果玩具宇宙；**Q萌不幼稚的两根支柱**＝驾驶员"童身大人派头"的反差萌 + 星舰"胖圆但仍是真飞行器"的载具感——**不靠冷酷高冷**（v1 路线作废）。
- **北极星（体验平滑不憋屈）转译**：界面读得清、反馈给得爽（Q弹动效/数值滚动/界面奖励飘字照旧）、不堆压抑战争感；卡关不憋屈靠星核质变"画面更爽"撑（总谱 V3 独占头部空间）。
- **画风锚点（唯一）**：`美术素材/锁样定稿/全局风格基准-糖果软萌温馨-15批版B抽3.png` + 同目录复现配方 md。**辅助质量标杆**：`敌方定稿参考-24批` 4 张（战斗 sprite 完成度标尺）、18 批 hub 抽1（界面金标准"鲜艳清澈透亮"）。

## A1 · 渲染语言与配色/材质（锁样配方逐条固化）

- **渲染**：软赛璐璐 + 糖果光泽高光；**圆润一体壳**（rounded one-piece smooth-shell）；**粗深同色系描边**（基于自身色加深，禁白描边、禁纯黑线堆）；**大色块、内缝极少**（内缝=邻区体色加深 20-30% 的同色系线，全身 ≤3 条宽结构缝）。
- **禁**（锁样 avoid 集固化）：铆钉 / 通风口 / 格栅 / 密集面板线 / 蜂窝纹 / 电路纹 / 碎拼贴组装感 / 乐高块 / 军武写实 / 机甲 Gundam / 暗黑赛博 / 血腥恐怖。
- **糖果主色盘**（锁样）：薄荷青 / 天空蓝 / 柠檬黄 / 糖果橙 / 奶油白 / 梦幻浅紫 / 软粉；主行动色高饱和暖橙（UI 侧走 B0）。**精确色值表**待批次实图取样后补。
- **甜暗色规范（29 批实证固化成法）**：一切"深色"单位一律**糖果调深色**——深蓝莓 / 深葡萄 / 深可可 / 黑加仑 / 黑莓紫，**禁金属黑、禁灰死色**（反派的"黑"=24 批黑莓紫标杆）。
- **分层铁律沿用**：文字/控件/光效由引擎渲染叠加，原画不画 UI 字、不画特效（记忆 `codex-art-spec-vs-paint`；攻击特效=总谱四件套引擎合成，**资产纯净体**）。

## A2 · 形象设计语言（对齐锁样 · v1"高冷酷帅"标尺作废）

- **驾驶员**：**2-2.5 头身 Q萌儿童体型，表情神态像大人一样沉稳自信——童身与大人派头的反差萌**（锁样句 + 22 批标准 + 附录B B0.8 暂定口径；正式头身锁样仍归 Ron）。表情=淡定沉稳、至多一抹极浅微笑（26 批治表情教训：删"自信坏笑/抬下巴"）；**禁**婴儿脸 / 卖萌嘟嘴 / 忧郁苦相。性格/服装按附录D 驾驶员真源逐人。
- **星舰**：**胖圆真飞船**——矮胖圆润、大圆鼓泡座舱、糖果一体壳；**两条硬约束（28 批拍板沿用）**：①别过火——仍是真飞行器、不是软圆球；②**五舰乃至 20 舰剪影必须各异**（身份词写死"是哪一艘"，绝不同一胖圆样）。**弱拟人、不给脸**（脸归驾驶员——铁律沿用；星舰一给脸卖萌就幼稚）。速度感/火力感靠**比例与部件**（大引擎/大炮管/环翼），**尖形状词全面禁用**（尖箭头 / 后掠刀翼 / 流线锋利 / 三棱锋刃——24-27 批实证反噬）。
- **呈现语言铁律（锁样记录 2026-07-05）**："驾驶员坐在敞开座舱内"不是本作语言，**禁止出现**——装配页=驾驶员立绘站舰旁（朝向公约 B0.8）；战斗=舰顶小头像；星舰本体不画人。**敌方例外**：敌单位可带小反派乘员（24 批定稿通则——小海盗坐船上是敌方的萌），我方不适用。
- **我方萌感代偿**：我方是"不给脸"的纯载具、情感分先天低于带小海盗的敌舰——靠 **舰顶驾驶员头像 + 全场最亮最甜的主色 + V0 呼吸层动效**（引擎火苗/悬浮/灯点，总谱 §1）扳回。

## A3 · 我方 20 舰 · 三层识别体系（全局语法 · 逐舰档案=附录D 星舰真源）

> 第一性：战斗里星舰**不移动、只转向开火**，战斗尺寸约 90-135px——一艘舰的"特色"只可能来自四样：**剪影、颜色、识别锚点、开火签名**。验收标尺=**35% 缩小三秒测试**：并排小图 3 秒内答出"哪型？哪舰？"。

- **第一层 · 敌我一眼分**：我方=糖果亮色 + 统一**星形徽章** + 机头朝上 + **青蓝引擎光**（全队统一）；敌方=族群色 + 各自徽记（骷髅/故障光/锈补丁/污染纹）+ 机头朝下 + 引擎光=族群色暗调。**铁律：我方永远是画面里最亮最甜的一队。**
- **第二层 · 定位型一眼分（五型剪影语法 + 尺寸阶 + 徽记色）**：

| 定位 | 剪影语法（身份词骨架） | 尺寸阶（战斗 sprite 相对基准） | 徽记色 |
|---|---|---|---|
| 突击 | **最小最紧凑**的糖果火箭：小身板 + 不成比例的大引擎，速度感靠比例不靠尖角 | 0.85（最小） | 糖果橙 |
| 护卫 | **最宽最厚**：盾形圆前脸、一身圆墩装甲 | 1.15（最大） | 天空蓝 |
| 炮击 | **一根大炮管当主角**：舰身=炮的底座 | 1.05 | 柠檬黄 |
| 支援 | **圆环 / 光翼 / 水滴 / 花瓣**：全队最柔和圆润 | 0.95 | 薄荷青 |
| 工程 | **方壳母舱工具感**：外挂机库/挂舱/天线/卷轴 | 1.05 | 梦幻紫 |

　徽记色以统一位置的小色带/小徽标呈现（具体孔位随俯视 sprite 批定稿）；尺寸阶为指导值、落地随格宽微调。
- **第三层 · 个体一眼分**：每舰五字段**视觉档案**（剪影读法 / 主·副色 / 识别锚点 / 开火签名 / 禁）＝`GDD-附录D-星舰真源.md` 逐舰；**全局配色律**：①同型 4 舰主色不撞 hue；②甜暗色规范（A1）；③**弹药颜色=本舰主色**（总谱 §2 颜色语法——舰的身份延伸到它的火力）。
- **敌人 4 族群**：造型基调归附录D 敌人真源；统一糖果渲染与线条规范（A1）；**星盗 4 单位=已定稿标杆**，后续族群身份词沿用其六段写法（族群 look 段 + 锁样锚点段 + 线条段 + 构图段 + 净底段 + 硬约束段）。
- **双套素材与朝向（沿用 · Ron 2026-06-25 拍板）**：我方星舰双套=① 3/4 精美立绘（装配/船坞/收藏）② 俯视 sprite（战斗·透明底·机头朝上）；**3/4 当设计母版、俯视照 3/4 派生**（同船跨视角必须对得上，**俯视必须保留视觉档案的识别锚点**）；驾驶员仅 3/4（战斗裁头像——**立绘必须"裁头可用"**，头像锚点/组徽环色=驾驶员真源视觉总则）；敌人仅俯视（机头朝下）；**星核/插件=展示图/图标 + 质变特效**（非战场单位——星核例外走"挂件"上战场=总谱 §2b；星核本体"糖果星核"物件语言=星核真源视觉总则；**插件不上战场**，命名=「XX插件」型号格式，图标=统一插芯底座〔底座色=槽位〕+具象微缩件+品质框=插件真源视觉与命名总则）；战斗只转向不位移。装配页立绘朝向公约=附录B B0.8。
- **升阶/升星不换舰体外观（Ron 2026-07-07 拍板④）**：20 舰各只出一套双视角素材；成长的"画面变强"**全走特效层**（星核质变/技能演出=总谱 V3），不做阶段皮肤、不翻倍素材量。
