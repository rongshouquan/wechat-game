## A0 · 总调性与北极星视觉转译

- **一句话画风（=全局唯一锁样）**：**糖果软萌温馨 · 软赛璐璐 + 糖果光泽**（第15批版B-抽3 · Ron 2026-07-05 锁）。明亮、暖、透亮的糖果玩具宇宙；**Q萌不幼稚的两根支柱**＝驾驶员"童身大人派头"的反差萌 + 星舰"胖圆但仍是真飞行器"的载具感——**不靠冷酷高冷**（v1 路线作废）。
- **北极星（体验平滑不憋屈）转译**：界面读得清、反馈给得爽（Q弹动效/数值滚动/界面奖励飘字照旧）、不堆压抑战争感；卡关不憋屈靠星核质变"画面更爽"撑（总谱 V3 独占头部空间）。

## A1 · 渲染语言与配色/材质（锁样配方逐条固化）

- **渲染**：软赛璐璐 + 糖果光泽高光；**圆润一体壳**（rounded one-piece smooth-shell）；**粗深同色系描边**（基于自身色加深，禁白描边、禁纯黑线堆）；**大色块、内缝极少**（内缝=邻区体色加深 20-30% 的同色系线，全身 ≤3 条宽结构缝）。
- **禁**（锁样 avoid 集固化）：铆钉 / 通风口 / 格栅 / 密集面板线 / 蜂窝纹 / 电路纹 / 碎拼贴组装感 / 乐高块 / 军武写实 / 机甲 Gundam / 暗黑赛博 / 血腥恐怖。
- **糖果主色盘**（锁样）：薄荷青 / 天空蓝 / 柠檬黄 / 糖果橙 / 奶油白 / 梦幻浅紫 / 软粉；主行动色高饱和暖橙（UI 侧走 B0）。**精确色值表**待批次实图取样后补。
- **甜暗色规范（29 批实证固化成法）**：一切"深色"单位一律**糖果调深色**——深蓝莓 / 深葡萄 / 深可可 / 黑加仑 / 黑莓紫，**禁金属黑、禁灰死色**（反派的"黑"=24 批黑莓紫标杆）。
- **分层铁律沿用**：文字/控件/光效由引擎渲染叠加，原画不画 UI 字、不画特效（记忆 `codex-art-spec-vs-paint`；攻击特效=总谱四件套引擎合成，**资产纯净体**）。

## A2 · 形象设计语言（对齐锁样 · v1"高冷酷帅"标尺作废）

- **驾驶员**：**2-2.5 头身 Q萌儿童体型，表情神态像大人一样沉稳自信——童身与大人派头的反差萌**（锁样句 + 22 批标准 + 附录B B0.8 暂定口径；正式头身锁样仍归 Ron）。表情=淡定沉稳、至多一抹极浅微笑（26 批治表情教训：删"自信坏笑/抬下巴"）；**禁**婴儿脸 / 卖萌嘟嘴 / 忧郁苦相。性格/服装按附录D 驾驶员真源逐人。
- **星舰**：**胖圆真飞船**——矮胖圆润、大圆鼓泡座舱、糖果一体壳；**两条硬约束（28 批拍板沿用）**：①别过火——仍是真飞行器、不是软圆球；②**五舰乃至 20 舰剪影必须各异**（身份词写死"是哪一艘"，绝不同一胖圆样）。**弱拟人、不给脸**（脸归驾驶员——铁律沿用；星舰一给脸卖萌就幼稚）。速度感/火力感靠**比例与部件**，**尖形状词全面禁用**（24-27 批实证反噬）。
- **★ 造型三版·"家族脸 + 每舰本色"（Ron 2026-07-07 拍板·33 批海报实证推翻二版制服路线）**：二版"舰队制服色（奶白+天蓝+金）"**作废**——实证=制服底和每舰本色打架、点缀补丁感、糖果味掉、反而更糊（33 批只突击/护卫像一队）。**根因诊断（王者/宝可梦第一性）**：统一 ≠ 配色统一（王者百英雄各配其色、宝可梦千只五颜六色，全靠"同款脸+同套工艺+中性舞台"统一，不靠同色）。新路线两层分离：
  - **统一层（血缘·锁死）**：① 画风工艺（软赛璐璐+糖果光泽+同色系描边+克制细节·锁样配方）② **标准星舰体（家族脸）**=所有 20 舰公共底模，见 A3 第二层 ③ 中性舞台（背景统一克制·A4 明亮律）。
  - **个性层（性格·放开）**：① **每舰回归本色**（撤制服·五颜六色 OK）② 剪影比例各异 ③ 职能装备件各异。
  - **约束少而精（救糖果感·教训 [[codex-art-spec-vs-paint]]）**：二版堆"骨架四件套+部件≤半身+制服"过多规格词→把 AI 从"画糖果原画"拽成"拼装机械件"。**改法=用正面底模代替负面规格**：把"标准星舰体"写成一艘本来就萌的糖果小飞船当底，每舰在底模上换色+加装备，既压住球化/枪化（底模兜着）又保糖果味（底模自萌）。
- **呈现语言铁律（锁样记录 2026-07-05）**："驾驶员坐在敞开座舱内"不是本作语言，**禁止出现**——装配页=驾驶员立绘站舰旁（朝向公约 B0.8）；战斗=舰顶小头像；星舰本体不画人（**泡舱内只透操作台剪影·40 批 Ron 去驾驶员人影、只留操作台剪影·非人**）。**敌方例外**：敌单位可带小反派乘员（24 批定稿通则——小星盗坐船上是敌方的萌），我方不适用。
- **我方萌感代偿**：我方是"不给脸"的纯载具、情感分先天低于带小星盗的敌舰——靠 **舰顶驾驶员头像 + 全场最亮最甜的主色 + V0 呼吸层动效**（引擎火苗/悬浮/灯点，总谱 §1）扳回。

## A3 · 我方 20 舰 · 三层识别体系（全局语法 · 逐舰档案=附录D 星舰真源）

> 第一性：战斗里星舰**不移动、只转向开火**，战斗尺寸约 90-135px——一艘舰的"特色"只可能来自四样：**剪影、颜色、识别锚点、开火签名**。验收标尺=**35% 缩小三秒测试**：并排小图 3 秒内答出"哪型？哪舰？"。

- **第一层 · 敌我一眼分（造型三版·Ron 2026-07-07）**：我方＝**统一"标准星舰体"家族脸＋机头朝上＋青蓝引擎光**（管"是一家"·**40 批 Ron 去星徽·舰身不画星星**）＋**每舰本色**（管"是自己"·撤制服）；敌方=族群色+各自徽记（骷髅/故障光/锈补丁/污染纹）+机头朝下+引擎光=族群色暗调。**铁律：我方永远是画面里最亮最甜的一队；一眼是一家（家族脸），二眼五职责，三眼哪一艘（本色+装备）。**
- **第二层 · 标准星舰体（家族脸=统一的核心手段·造型三版）**：**所有 20 舰共享的公共底模**——胖圆一体糖果壳 / 一颗**椭圆泡形座舱**（有色玻璃·**舱内只透操作台剪影**·40 批 Ron 去人影·**这是"眼睛"级家族识别符，每舰必备同款**）/ 一对对称短翼或引擎舱 / 尾部青蓝喷口 / 糖果光泽渲染 + 克制内缝。**它自己就是一艘萌萌的糖果小飞船**（对齐锁样海报里我方主力舰的胖圆比例）。类型差异＝在这副底模上**改比例 + 挂装备件**，**不丢底模四件、不让装备件压过舰体**（软性口径：仍一眼是"带了装备的飞船"、不是"会飞的炮/盾/球"）：

| 定位 | 本质（Ron 2026-07-07 口径） | 比例配方（底模上调整） | 标志装备件（不压过舰体） | 尺寸阶 | 徽记色 |
|---|---|---|---|---|---|
| 突击 | 闪电战、灵活难缠 | 机身最小最窄、双大引擎占比高、前倾姿态 | 轻武器（短炮/光球/锥刺） | 0.90 | 糖果橙 |
| 护卫 | 厚实敦厚、坦克 | 机身最短最宽、翼短粗、装甲厚 | 光滑透明钢板（smooth transparent steel plates·40批 Ron 定·非盾板） | 1.10 | 天空蓝 |
| 炮击 | 强火力压制、一看就猛 | 标准舰身 | **机头主炮**（装机头前方·非背驮·39批）+弹舱鼓包 | 1.00 | 柠檬黄 |
| 支援 | 治疗增益 | 全队最圆润柔和 | 翼尖弧光环片/光雾喷口/能量旗 | 0.95 | 薄荷青 |
| 工程 | 多功能（召唤/控制） | 短宽工作艇 | 外挂工具件（机库口/天线/线轴/瞭望塔） | 1.00 | 梦幻紫 |

　徽记色以统一位置的小色带/小徽标呈现（孔位随俯视 sprite 批定稿）；尺寸阶 0.90-1.10（护卫最大但不球化）。**徽记色=小面积色标**（不是整舰主色），与第三层每舰本色并存不冲突。
- **第三层 · 个体一眼分（每舰本色·造型三版）**：每舰五字段**视觉档案**（剪影 / 本色〔主色+副色·各舰自成一格〕/ 识别锚点 / 开火签名 / 禁）＝`GDD-附录D-星舰真源.md` 逐舰；**全局配色律**：①**每舰回归本色、五颜六色 OK**（撤制服·统一靠家族脸不靠同色）；②同型 4 舰主色互异（hue 错开）；③甜暗色规范（A1·深色舰走糖果调深莓/深葡萄，非金属黑）；④**弹药颜色=各舰开火签名所记发光色**（总谱 §2——舰的身份延伸到它的火力）。
- **敌人 4 族群**：造型基调归附录D 敌人真源；统一糖果渲染与线条规范（A1）；**星盗 4 单位=已定稿标杆**，后续族群身份词沿用其六段写法（族群 look 段 + 锁样锚点段 + 线条段 + 构图段 + 净底段 + 硬约束段）。
- **双套素材与朝向（沿用 · Ron 2026-06-25 拍板）**：我方星舰双套=① 3/4 精美立绘（装配/船坞/收藏）② 俯视 sprite（战斗·透明底·机头朝上）；**3/4 当设计母版、俯视照 3/4 派生**（同船跨视角必须对得上，**俯视必须保留视觉档案的识别锚点**）；驾驶员仅 3/4（战斗裁头像——**立绘必须"裁头可用"**，头像锚点/组徽环色=驾驶员真源视觉总则）；敌人仅俯视（机头朝下）；**星核/插件=展示图/图标 + 质变特效**（**均不作常驻战场物件**——星核战斗中只在头顶 HUD 一个小星核宝石图标、发动瞬间从图标位置炸开质变演出＝总谱 §2b〔"挂件"路线 33 批实证突兀，作废〕；星核本体"糖果星核"物件语言=星核真源视觉总则，用于展厅/图标/发动演出；**插件不上战场**，命名=「XX插件」型号格式，图标=统一插芯底座〔底座色=槽位〕+具象微缩件+品质框=插件真源视觉与命名总则）；战斗只转向不位移。装配页立绘朝向公约=附录B B0.8。

### A6.2 · 体积预算总账（v0 草案 · 与工程总控对账后转正）

- **平台红线**：主包 **3.8MB 警戒 / 4.0MB 阻断**（现主包 2823KB → 全部余量约 1068KB）；整包按 **≤20MB 保守规划**（当期官方上限，首批接入时与工程总控核准）。
- **配额草案**：主包美术（UI 皮肤+图标+字体）**≤600KB**，其余余量留工程增长 ｜ 分包·界面场景 **≤3MB** ｜ 分包·单位资产（20 舰双视角＋敌方＋驾驶员立绘头像）**≤4MB** ｜ 分包·特效图集＋背景动效 **≤1.5MB**（其中背景动效 ≤300KB＝总谱 §4 红线）。
- **记账规则**：每资产接入记 预计/实际 KB＋授权来源＋`.meta` 状态（沿用）；《全游戏视觉表》每条带"预计体积"列、分批汇总对本表配额；**超额处理顺序＝压缩参数 → 降分辨率 → 图集重排 → 最后才谈扩配额**。

### A6.3 · 字体硬门槛

- 首选**系统字体**（0 体积）；若上自定义字体**必须子集化**（只打包用到的字）并占主包美术配额——中文全量字体动辄 10MB 级，**禁整包引入**。UI 皮肤阶段终拍。

### A6.4 · 商用许可硬门槛

- 出图模型/放大器/一切素材源，用前**逐个查商用许可**（台账＝总纲 §2）；非商用许可的产物只做内部预览，**禁入包、禁对外物料**（总纲铁律A2）。
