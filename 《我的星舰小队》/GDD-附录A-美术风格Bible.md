## A0 · 总调性与北极星视觉转译

- **一句话画风**：明亮、暖、协调的 **painted 成品质感**「软萌星港」+ **Q萌但高冷酷帅的成人卡通**角色，不幼稚。
- **北极星（体验平滑不憋屈）的视觉转译**：界面读得清、反馈给得爽（Q弹动效/数值滚动/奖励飘字）、不堆压抑战争感；卡关不憋屈靠"画面更爽的质变"撑（星核质变要肉眼可见变炫）。
- **题材基调**（GDD §1.1⑥）：轻科幻 + Q萌拟人 + 幽默热闹，明亮可爱不冷峻。

## A1 · 画风基准与配色/材质

- **画风基准（Ron 2026-06-24 海选拍板）= `2026-06-18 软萌星港` 那套 painted 风**（基地海岛 / Q萌驾驶员星空战斗 / 抽补给 / 战前备战）+ `2026-06-20`（打捞/商人/星空宝库）。海选淘汰：纯扁平UI稿、深空冷酷霓虹（C）。
- **配色**：底色明亮（浅蓝/奶白/暖灰），主行动色高饱和暖橙（出战/确认），辅助功能色清爽蓝绿紫；撞色不刺眼。**精确色值表**待补（A 阶段细化）。
- **材质**：圆角面板 + 暖色光影 + 轻盈发光点缀；金属/玻璃/能量要**有层次质感**（成人卡通），**少糖果塑料感**。
- **教训（记忆 `codex-art-spec-vs-paint`）**：出成品图=**放开当原画画**，别做扁平排版稿、别逐个标控件名（会拽平）；UI 精确布局走附录B、与 painted 原画**分层合成**。

## A2 · 形象设计语言（"成人卡通·不幼稚"铁律 · 本session 锁定）

> 解"V2 太幼稚"的根因。**这是全游戏所有角色/星舰造型的统一标尺。**

- **驾驶员**：**3~3.5 头身**、五官精致有个性、**高冷酷帅有型**（冷峻眼神、克制表情）；**禁** 2头身大头娃娃 / 婴儿脸 / 卖萌。性格/服装按**附录D 驾驶员真源**逐人。
- **星舰**：**圆润但仍是"战机/载具"**——金属机械结构、火力与能量细节、流线锋利；**弱拟人、不给脸**（脸交给驾驶员，附录D 铁律）；**星舰一给脸卖萌就幼稚**。定位造型按**附录D 星舰真源**（5 定位型：突击/护卫/炮击/支援/工程）。
- **总气质**：Q萌 + **高冷 + 冷酷**的成人卡通（参考"暴雪/皮克斯短片的成熟可爱"，不是低龄贴纸）。
- **🔒 形象基准锚点（A-0 已定 · Ron 2026-06-25 过及格线）**：以 **赤焰（星舰）+ 赫炎（驾驶员）** 单舰装配整屏效果图定死「高冷酷帅不幼稚」标尺 = **`2026-06-25_我的星舰小队主角形象基准锁定/最终成品/形象基准-单舰装配-take1-仅改形象.png`**（赫炎=红发酷帅、3+头身、冷峻有型；赤焰=流线红色战机、金属机械火力细节、不给脸）。后续所有单位造型对齐此锚点 + 附录D。可再高冷一档为加分项、非必须。
  - ⚠️ **只锁"形象 + 画风"，不锁图上文字**：该图的中文 UI 叠字乱/错位/按钮重叠 = **AI 出图天生画不准文字**（中文+多控件尤甚），**属正常、不修、不重出**。最终 UI 文字/控件由**引擎渲染**（清晰对齐），见 A1 分层铁律与记忆 `codex-art-spec-vs-paint`。

- **🔒 素材视角与双套规则（Ron 2026-06-25 拍板）**：
  - **我方星舰 = 双套**：① **3/4 精美立绘**（展示用：装配/船坞/收藏，撑收藏养成的"好看"）② **俯视图 sprite**（战斗用：透明底、机头朝上为基准、可整体转向）。
  - **驾驶员 = 仅 3/4 立绘**（战斗里是头顶头像，由立绘裁头）；**敌人 = 仅俯视 sprite**（不进收藏，机头朝下为基准，也转向）；**星核/插件 = 展示图/图标 + 质变特效**（非战场"船"，特效复用 A5 弹种）。
  - **朝向统一**：我方机头朝上（朝敌人）、敌方机头朝下；战斗"只转向不位移"（对齐附录D 固定阵位）。
  - **一致性工作流（关键）**：同一艘船的两套**配对产出**，且 **3/4 当设计母版、俯视照 3/4 派生**（避免"先全俯视再全 3/4"导致同船两视角对不上）。Codex 出俯视时把该船 3/4 当参考图喂入。
