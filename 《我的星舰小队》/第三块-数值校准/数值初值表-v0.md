## 3. 战力公式 v0（B1 §1 骨架 + S5.5 计入项落定）

```
单舰战力 = ( 阶级基值 × (1 + 0.08×(舰等级−1)) + 插件合计 ) × 驾驶员系数 + 星核120
驾驶员系数 = 星级系数[1.00/1.08/1.18/1.30/1.45] × (1 + 0.01×驾驶员等级)
队伍战力 = Σ上阵5舰 × (1 + 研究塔1%/级 + 展厅收藏加成·上限10%)
```

## 5. 玩家策略假设全文（尺子系统误差的主要来源 · 透明化）

四档共用同一固定优先级贪心策略，只差档位参数：

1. **回港领取**：离线+巡逻按离开时长累积（居住舱存储上限截断）；看广告档必用 #1 报告翻倍。

4. **推主线优先**（有效战力=战力×(1+试错折算)）；看广告档 #3 固定翻倍+#4 再选一用在当日最后一关。

## 11. 给任务单②（资源循环第一原理审视）的可疑循环线索

1. **星贝死水**：普通档毕业结余 ~11.6 万——券日限40+加速券+篮吸不完，溢出星矿回收还在灌。方向：中后期星贝 sink 扩容（加速券用量随信标池涨/宝库星贝区/高价稀有货）或回收比调贵。
2. **非主力碎片沉淀**（舰/员各 ~3,500）：归属集中度 0.34 下 2/3 碎片喂 bench。审视：bench 有没有真出口（悬赏词缀鼓励多定位型阵容=苗头，但战力/推进只认 top5）。
3. **信标积压**：普通毕业剩 ~330 普通信标，供给>消化力（2队×2趟）。审视：打捞港 lv4/lv7 队列解锁节奏与加速券定位是否接得住。
4. **打捞是载重梁**：单源清零→+64 天（最大单源）；通用碎片 1:1 定向转换权重极高。审视：转换比、以及非酋信标断供时碎片线抖动（欧非带宽的主要来源）。
5. **星核通胀苗头**：三轮砍量后普通毕业仍 ~24 颗（16 种设计·同款可多装吃掉部分）；星核碎片渠道随星域系数放大是主因。审视：frag 渠道是否去星域缩放改固定滴流（守"第一稀缺品"人设）。
6. **广告#1 一家独大**：回港翻倍在满广告收益中占大头；普通 2 广告画像下它必占其一。审视：十点位"少而肥"vs 实际肥瘦不均，#5/#8/#9 价值偏弱。
7. **毕业后星矿死币**：建筑封顶后星矿唯一出口=回收。审视：毕业后星矿 sink（长线路线图问题）。
8. **三选一"点名"损耗**：非主力碎片是三选一机制的天然损耗。审视：是否给"已拥有/主力单位"加点名权重（强平衡杠杆，动前要过体验判断）。
