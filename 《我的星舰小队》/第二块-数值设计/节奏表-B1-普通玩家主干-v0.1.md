## 1. 战力公式 v0.1

```
单舰战力 = ( 星舰基础 + 插件合计 ) × 驾驶员系数 + 星核加成
队伍战力 = Σ 上阵各舰战力
```
- **星舰基础** = 阶级基值 × (1 + 0.08 × (星舰等级 − 1))
  - 阶级基值：C=100 / B=160 / A=250 / S=380 / SS=550（每阶约 ×1.5）
- **插件合计** = 三槽之和；精良=15 / 优秀=35 / 传奇=70
- **驾驶员系数**：1★=1.00 / 2★=1.08 / 3★=1.18 / 4★=1.30 / 5★=1.45
- **星核加成** = +120 /（装核的船）；星核的“规则质变”价值不计入战力（呼应根文件“战力够也可能没赢”）

## 2. 养成消耗曲线 v0.1（形状，先定形不抠值）
- 星舰升级：合金消耗 ≈ 50 × 等级^1.3（越后越贵）；等级上限受船坞等级限制。
- 星舰升阶 C→B→A→S→SS：星舰碎片 ≈ 20 / 40 / 80 / 160（每阶翻倍）。
- 驾驶员升级：驾驶记录 ≈ 40 × 等级^1.2。
- 驾驶员升星 1→5★：驾驶员碎片 ≈ 20 / 40 / 80 / 160。
- 插件合成：3 同类精良→1 优秀；3 同类优秀→1 传奇（=9 精良→1 传奇）。
- 建筑升级：星矿为主，逐级上升；建筑封顶（10 级）后星矿溢出 → 回收为星贝。

## 3. 关卡战力墙 v0.1（“更陡”：星域末 / Boss 起墙制造卡点）
- 普通关推荐战力随节点平滑上升；**每个星域的最后 2-3 关 + Boss 把墙抬到当前战力之上**，形成“卡点”——逼玩家养成 1-2 天再突破（精英 ×1.3、Boss ×1.6 于该段普通关均值）。
- 按普通玩家战力曲线反推的关键节点墙（4 星域 × 约 15 节点 = 60 节点示意）：

| 进度节点 | 推荐战力（普通关） | 对应时点 |
|---|---|---|
| 星域1 中（n≈8） | ~350 | D1-2 |
| 星域1 Boss（n≈15） | ~750（含 ×1.6 spike，需首核破） | D3 |
| 星域2 中（n≈22） | ~1,400 | D5 |
| 星域2 Boss（n≈30） | ~2,700 | D7 |
| 星域3 Boss（n≈45） | ~6,000 | D14 |
| 星域4 Boss / 毕业（n≈60） | ~10,800 | D21 |

> “更陡”体现：星域 Boss 墙明显高于当时巡航战力，必须靠“领离线 + 升阶 + 新核”攒过——这正是“卡关→渴望→变强→爽推”的节拍。

## 5. 星核获取时间线（普通玩家，验证“D21 ~5 核”）
- D3：核#1 = 第一个 Boss「星核宝藏」三选一（首核，破 Boss 的钥匙）。
- D7：核#2 = 第一次扩张宝藏「自选」。
- ~D10：核#3 = 星核碎片随机合成首颗（碎片约每 5-7 天攒够一颗）。
- D14：核#4 = 第二次扩张宝藏「随机三选一」。
- ~D18：核#5 = 碎片合成第二颗 或 星空宝石首次定向兑换。
- → 普通玩家约 **D18-21 凑齐 5 核**，与目标“D21 接近/形成 5 核”一致 ✓（重度更快）。
- 星空宝石定向兑换约每 10-14 天一颗，作为“想要哪颗就熬哪颗”的慢兜底，不抢随机渠道节奏。

## 6. 资源是否够用（普通玩家，方向性）
- **软货币**：主线每战必给 + 基地离线（随最高星域上升）。养成消耗曲线（§2）按“略低于产出”设，使升级/升阶始终跟得上巡航墙、卡点处需主动领离线补一波——即“平滑不憋屈”的来源。
- **补给券**：主线/Boss 首通 + 3天/7天活动 + 赞助补给（广告，baseline 不算）。前 ~10-14 天足够凑齐大部分本体（6-8 驾驶员 / 10-12 星舰），之后转“屯券”，要升阶/出新角色再抽。
- **碎片**：通用碎片（活动）+ 专属碎片（抽卡重复）支撑上表的升阶/升星态。
- **星核碎片 / 星空宝石**：按 §5 时间线滴流，靠合成成本/兑换价压住节奏。

## 7. 验证结论
- ✅ **循环转得起来**：战力曲线与“更陡”的墙能对齐到 D3 进域2 / D7 进域3 / D14 进域4 / D21 毕业+5核。
- ✅ **卡点平滑不憋屈**：卡点集中在星域末/Boss，破点靠“领离线 + 升阶 + 新核”，是“攒一波就过”的短闭环，不是干等。
- ✅ **不看广告成立**：本主干即 baseline，广告只加速。
- ✅ **星核作中期爽点引擎**：每 ~5-7 天有新核进账，卡 Boss 时恰有新核救场。

## 8. 按星核设计支柱校正（Ron 拍板，纠正我此前两条误判）
1. **星核战力加成 = 固定值（不改百分比）。** 前期就是要“拿到核变强很多”的质变感——这正是星核稀缺与爽的理由，固定值让早期一核就是大跳。**关键：星核的“质变感”主要来自规则效果（视觉 + 机制改写，不计入战力），固定战力加成只是次要；后期不靠“单核数值越堆越高”，靠搭配 + 真正极品 / 限定核。**
2. **不放慢星核获取速度。** 普通玩家 ~D18 凑齐 5 核没问题——“获得星核本身要有流速”才撑得起中期爽点引擎（隔三差五新玩具）。**真正该稀缺、该慢的是“极品核 / 宝库限定核 / 指定那一颗”**：靠星空宝石定向兑换（约每 10-14 天）+ 顶级更长保底拉长。**普通 5 核 ≠ 极品 5 核**，两件事分开。
3. 其余具体常数（每级加成、掉率、价格）都是 v0.1，待原型校准。

## 9. 四类玩家派生（相对普通玩家）

**关键洞察：时间门控是天然限速器。** 离线产出有上限（~1.5 天）、打捞有计时、周期活动按 3/7 天结算——这些资源不随“多玩几小时”增加。所以肝帝**不能无限甩开**普通：多出来的活跃时间能多推几关、多养成，但卡在战力墙时仍要等时间门控的资源。于是四类差距被压缩，而非线性放大。

进度速度系数（相对普通=1.0，已被时间门控压缩）：**轻度 ~0.75 / 普通 1.0 / 重度 ~1.3 / 肝帝 ~1.5**。

| 类型 | 主线毕业 | 凑齐 5 核（普通核） | 对照 §2 目标 |
|---|---|---|---|
| 轻度 | ~D28 | ~D24 | D28 毕业/接近 + 3-5 核 ✓ |
| 普通 | ~D21 | ~D18 | D21 毕业/接近 + 接近 5 核 ✓ |
| 重度 | ~D14-16 | ~D14 | D14 挑战最终 Boss、D21 5 核 ✓（更早，可接受） |
| 肝帝 | ~D14 | ~D12 | D7 进最终星域深处、D14 毕业 ✓ |

- **极品 / 限定核对所有类型都慢**（星空宝石定向兑换 ~10-14 天/颗 + 顶级更长保底）。普通 5 核 ≠ 极品 5 核——所以肝帝凑齐 5 核后，仍要长期追极品/限定核。这是星核作长线追求的关键。
- 诚实提醒：50-70 节点下，**重度/肝帝约 D14 前后就吃完主线** → 进入“毕业后”（追极品核 / 拔高 / 等后续内容）。这是首发内容量的必然，靠 §15 路线图（更多关卡/竞技场）+ 极品核长尾接住。属已知、可接受。
