## 一、插件词条种子（按槽位 × 品质）

**品质定位**（v1.0 §5.3 已定）：精良 = 单一清晰效果·数值温和 / 优秀 = 数值更强 / 传奇 = 强数值 + 一条额外小效果。

**按槽位的词条方向（种子例，非穷举）**：
- **武器槽**：+伤害% / 暴击率·暴击伤害 / 攻速 / 穿透 / 对小怪加伤 / 对 Boss 加伤 / 溅射。
- **技能(CD)槽**：−CD% / 开局加速首发 / 技能伤害+ / 技能范围+ / 触发概率+。
- **战术槽**：破盾值 / 减伤·护盾 / 控制时长 / 保护·嘲讽 / 回血 / 标记·易伤。

> 对照 v1.0 §5.5 已框定的定向词条：暴击率、暴击伤害、破盾、技能急速/缩CD/首发加速、治疗强度、控制时长/抗性、对小怪或对 Boss 加伤——上面的种子比 §5.5 多了 攻速/穿透/溅射/技能范围/触发概率/嘲讽/标记易伤 等方向，第二块定词条库时可斟酌纳入。

**合成**（v1.0 §5.3 已定，列此便于一处看全）：消耗 N 个**同槽位类型同品质** → 1 个**高一阶同类**；词条在该槽位类型内**随机**（定向到槽位、不定向到词条，留小赌点）。起步比例 3 精良→1 优秀、3 优秀→1 传奇（=9 精良→1 传奇，可调）。

## 二、四层串联种子例（导弹舰，把四层串起来看怎么不重叠）

- **星舰**（导弹舰）：自带技能「齐射」（CD技：发一排导弹）+ 被动「装填」（普攻概率多打一发）。升阶 → 解锁第二技能 / 强化齐射。
- **驾驶员·突袭手** 能力「猎杀后排」：改写齐射目标 → 优先锁后排；升星 → 对后排额外 +X% 伤害。
- **驾驶员·炮手** 能力「过热」：持续输出叠加伤害；升星 → 提高叠层上限。
- **插件**（武器·传奇）：+暴击率，暴击时溅射（纯数值 + 小效果）。
- **星核**（小太阳）：把「齐射」质变成「发射一颗吸附敌人后爆炸的小太阳」（规则级质变）。

→ 同一艘船：**星舰给动作、驾驶员决定打谁/怎么打、插件补数值、星核变身**，四层不重叠。第二块设计具体技能/能力/词条时，用这个"串联不重叠"的样子做自检标尺。
